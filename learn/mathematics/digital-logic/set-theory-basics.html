1) What is a set?

A set is a collection of distinct objects called elements. Sets are usually denoted by capital letters; elements by lowercase letters or numbers.

  • Roster form: A = {1, 2, 3} — list elements
  • Set-builder: B = {x | x is an even integer} — rule for membership
  • Empty set: ∅ or {} — no elements
  • Universal set: U — all objects under discussion in a given context
Key idea: Order does not matter: {1, 2, 3} = {3, 1, 2}. Duplicates are ignored: {1, 1, 2} = {1, 2}.

Set operations mirror Logic Gates and Boolean Algebra — intersection is AND, union is OR, complement is NOT.

2) Membership and subsets

Membership

  • x ∈ A — x is an element of A
  • x ∉ A — x is not in A

Subsets

  • A ⊆ B — every element of A is in B
  • A ⊂ B — A ⊆ B and A ≠ B (proper subset)
  • A = B iff A ⊆ B and B ⊆ A
Worked example: If A = {2, 4, 6} and B = {2, 4, 6, 8}, then A ⊂ B. The empty set is a subset of every set: ∅ ⊆ A for any A.

In Predicate Logic, A ⊆ B is written ∀x (x ∈ A → x ∈ B).

3) Union and intersection

Union collects elements in either set. Intersection collects elements in both.

OperationNotationDefinitionLogic gate
UnionA ∪ B{x | x ∈ A ∨ x ∈ B}OR
IntersectionA ∩ B{x | x ∈ A ∧ x ∈ B}AND
Worked example: A = {1, 2, 3}, B = {3, 4, 5}
A ∪ B = {1, 2, 3, 4, 5}
A ∩ B = {3}

Properties

  • Commutative: A ∪ B = B ∪ A
  • Associative: (A ∪ B) ∪ C = A ∪ (B ∪ C)
  • Identity: A ∪ ∅ = A, A ∩ U = A

Disjoint sets

A and B are disjoint when A ∩ B = ∅ — they share no elements.

4) Complement and set difference

The complement of A (relative to universal set U) is everything in U not in A. Set difference removes one set from another.

OperationNotationDefinition
ComplementA' or Aᶜ{x ∈ U | x ∉ A}
DifferenceA − B or A \ B{x | x ∈ A ∧ x ∉ B}
Worked example: U = {1, 2, 3, 4, 5}, A = {2, 4}
A' = {1, 3, 5}
If B = {1, 2, 3}, then A − B = {4} and B − A = {1, 3}.

Note: A − B ≠ B − A in general. Complement requires a defined universal set.

5) Venn diagrams

A Venn diagram represents sets as overlapping regions inside a rectangle (the universal set). Shading a region visualises the result of set operations.

How to read them

  • Each circle = one set
  • Overlap = intersection
  • Combined shaded circles = union
  • Region outside a circle = complement (within U)

Example regions

A ∩ B — lens-shaped overlap

A ∪ B — both entire circles

A ∩ B' — part of A excluding overlap

(A ∪ B)' — everything outside both circles

Strategy: For complex expressions like (A ∪ B) ∩ C', shade step by step: first A ∪ B, then restrict to complement of C.

6) De Morgan's laws (set form)

De Morgan's theorems in Boolean algebra have direct set-theoretic partners. They describe how complement interacts with union and intersection.

De Morgan (sets):
(A ∪ B)' = A' ∩ B'
(A ∩ B)' = A' ∪ B'

Generalised to n sets:
(A₁ ∪ A₂ ∪ … ∪ Aₙ)' = A₁' ∩ A₂' ∩ … ∩ Aₙ'
(A₁ ∩ A₂ ∩ … ∩ Aₙ)' = A₁' ∪ A₂' ∪ … ∪ Aₙ'

Worked example: Verify with U = {1,…,6}, A = {1,2,3}, B = {3,4,5}.
A ∪ B = {1,2,3,4,5}, so (A ∪ B)' = {6}.
A' = {4,5,6}, B' = {1,2,6}, so A' ∩ B' = {6}. ✓

Logic form: ¬(p ∨ q) ≡ ¬p ∧ ¬q — see Propositional Logic.

7) Distributive and absorption laws

Set algebra shares the same structural laws as Boolean algebra, which simplifies circuit and proof work.

LawSet formBoolean form
DistributiveA ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)A · (B + C) = A·B + A·C
DistributiveA ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)A + B·C = (A+B)·(A+C)
AbsorptionA ∪ (A ∩ B) = AA + A·B = A
AbsorptionA ∩ (A ∪ B) = AA · (A + B) = A
Worked example — simplify: (A ∩ B) ∪ (A ∩ B')
Distribute factorisation in reverse: A ∩ (B ∪ B') = A ∩ U = A.

  4. Apply De Morgan to simplify (A' ∪ B)'.
  5. Shade A ∩ (B ∪ C) on a three-circle Venn diagram — which regions count?
