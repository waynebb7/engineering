Formulae & relationships

\(A \cup B = \{x \mid x \in A \lor x \in B\}\)
Union — elements in A or B (logic OR).
\(A \cap B = \{x \mid x \in A \land x \in B\}\)
Intersection — elements in both A and B (logic AND).
\(A' = \{x \in U \mid x \notin A\}\)
Complement of A relative to universal set U (also written \(A^c\)).
\(A \setminus B = A \cap B'\)
Set difference — elements in A but not in B.
\(A = B \iff A \subseteq B \land B \subseteq A\)
Set equality — same elements iff each is a subset of the other.
\(A \subset B \iff A \subseteq B \land A \neq B\)
Proper subset — A is contained in B but not equal to B.
\(A \cup \emptyset = A\), \(A \cap U = A\)
Identity laws with empty set and universal set.
\((A \cup B)' = A' \cap B'\)
De Morgan's law — complement of a union.
\((A \cap B)' = A' \cup B'\)
De Morgan's law — complement of an intersection.
\(A \cap (B \cup C) = (A \cap B) \cup (A \cap C)\)
Distributive law — intersection over union.
\(A \cup (A \cap B) = A\), \(A \cap (A \cup B) = A\)
Absorption laws (set form).

Symbol names in formulae

\(A\), \(B\), \(C\)
Called: capital letters. Sets — collections of elements.
\(U\)
Called: capital U. Universal set — all objects under discussion in a given problem.
\(x\)
Called: lowercase x. A typical element ranging over members of a set.
\(\{ \}\)
Called: set braces. Enclose listed or rule-defined elements, e.g. \(\{1,2,3\}\) or \(\{x \mid \text{condition}\}\).
\(\emptyset\)
Called: empty set. The set with no elements (also written \(\{\}\)).
\(\in\)
Called: element-of. \(x \in A\) means x is a member of set A.
\(\notin\)
Called: not-in. \(x \notin A\) means x is not a member of A.
\(\subseteq\)
Called: subset (or equal). Every element of A is also in B.
\(\subset\)
Called: proper subset. \(A \subset B\) means \(A \subseteq B\) and \(A \neq B\).
\(\neq\)
Called: not equal. Used when two sets are not the same, e.g. in the proper-subset definition.
\(\cup\)
Called: union. Combines elements from either set (logic OR).
\(\cap\)
Called: intersection. Keeps elements common to both sets (logic AND).
\(A'\), \(A^c\)
Called: A prime, A complement, or A superscript c. Elements of U not in A (logic NOT).
\(\setminus\)
Called: set minus. Set difference \(A \setminus B\) (also written \(A - B\)).
\(\lor\)
Called: logical OR. Used in set-builder notation for union (\(x \in A \lor x \in B\)).
\(\land\)
Called: logical AND. Used in set-builder notation for intersection (\(x \in A \land x \in B\)).
\(\mid\)
Called: such that. Separates the variable from the membership rule in set-builder notation (\(\{x \mid \text{condition}\}\)).
\(\iff\)
Called: if and only if. States equivalence, e.g. for set equality or proper subset.

Key terms & phrases

set
A collection of distinct objects called elements.
elements
The individual objects that belong to a set.
roster form
Listing all elements inside braces, e.g. \(A = \{1, 2, 3\}\).
set-builder notation
Describing a set by a rule, e.g. \(B = \{x \mid x \text{ is an even integer}\}\).
empty set
The set with no elements, written \(\emptyset\) or \(\{\}\); a subset of every set.
universal set
All objects under discussion in a given context, denoted \(U\); needed to define complements.
membership
Whether an element belongs to a set (\(x \in A\)) or not (\(x \notin A\)).
subset
\(A \subseteq B\) — every element of A is also in B (A may equal B).
proper subset
\(A \subset B\) — A is contained in B but is not equal to B.
set equality
\(A = B\) when \(A \subseteq B\) and \(B \subseteq A\).
union
Combines elements from either set; corresponds to the OR logic gate.
intersection
Keeps elements common to both sets; corresponds to the AND logic gate.
complement
Relative to \(U\), everything in U not in A; written \(A'\) or \(A^c\).
set difference
Elements in A but not in B; \(A \setminus B\) or \(A \cap B'\).
disjoint sets
Sets with no elements in common: \(A \cap B = \emptyset\).
Venn diagram
Overlapping regions inside a rectangle (\(U\)) used to visualise set operations.
De Morgan's laws
\((A \cup B)' = A' \cap B'\) and \((A \cap B)' = A' \cup B'\); complements swap union and intersection.
distributive laws
Intersection distributes over union and vice versa, mirroring Boolean algebra.
absorption laws
\(A \cup (A \cap B) = A\) and \(A \cap (A \cup B) = A\).
commutative, associative, identity
Union and intersection reorder freely; \(\emptyset\) and \(U\) act as identities for \(\cup\) and \(\cap\).

1) What is a set?

A set is a collection of distinct objects called elements. Sets are usually denoted by capital letters; elements by lowercase letters or numbers.

  • Roster form: A = {1, 2, 3} — list elements
  • Set-builder: B = {x | x is an even integer} — rule for membership
  • Empty set: ∅ or {} — no elements
  • Universal set: U — all objects under discussion in a given context
Key idea: Order does not matter: {1, 2, 3} = {3, 1, 2}. Duplicates are ignored: {1, 1, 2} = {1, 2}.

Set operations mirror Logic Gates and Boolean Algebra — intersection is AND, union is OR, complement is NOT.

2) Membership and subsets

Membership

  • x ∈ A — x is an element of A
  • x ∉ A — x is not in A

Subsets

  • A ⊆ B — every element of A is in B
  • A ⊂ B — A ⊆ B and A ≠ B (proper subset)
  • A = B iff A ⊆ B and B ⊆ A
Worked example: If A = {2, 4, 6} and B = {2, 4, 6, 8}, then A ⊂ B. The empty set is a subset of every set: ∅ ⊆ A for any A.

In Predicate Logic, A ⊆ B is written ∀x (x ∈ A → x ∈ B).

3) Union and intersection

Union collects elements in either set. Intersection collects elements in both.

OperationNotationDefinitionLogic gate
UnionA ∪ B{x | x ∈ A ∨ x ∈ B}OR
IntersectionA ∩ B{x | x ∈ A ∧ x ∈ B}AND
Worked example: A = {1, 2, 3}, B = {3, 4, 5}
A ∪ B = {1, 2, 3, 4, 5}
A ∩ B = {3}

Properties

  • Commutative: A ∪ B = B ∪ A
  • Associative: (A ∪ B) ∪ C = A ∪ (B ∪ C)
  • Identity: A ∪ ∅ = A, A ∩ U = A

Disjoint sets

A and B are disjoint when A ∩ B = ∅ — they share no elements.

4) Complement and set difference

The complement of A (relative to universal set U) is everything in U not in A. Set difference removes one set from another.

OperationNotationDefinition
ComplementA' or Aᶜ{x ∈ U | x ∉ A}
DifferenceA − B or A \ B{x | x ∈ A ∧ x ∉ B}
Worked example: U = {1, 2, 3, 4, 5}, A = {2, 4}
A' = {1, 3, 5}
If B = {1, 2, 3}, then A − B = {4} and B − A = {1, 3}.

Note: A − B ≠ B − A in general. Complement requires a defined universal set.

5) Venn diagrams

A Venn diagram represents sets as overlapping regions inside a rectangle (the universal set). Shading a region visualises the result of set operations.

How to read them

  • Each circle = one set
  • Overlap = intersection
  • Combined shaded circles = union
  • Region outside a circle = complement (within U)

Example regions

A ∩ B — lens-shaped overlap

A ∪ B — both entire circles

A ∩ B' — part of A excluding overlap

(A ∪ B)' — everything outside both circles

Strategy: For complex expressions like (A ∪ B) ∩ C', shade step by step: first A ∪ B, then restrict to complement of C.

6) De Morgan's laws (set form)

De Morgan's theorems in Boolean algebra have direct set-theoretic partners. They describe how complement interacts with union and intersection.

De Morgan (sets):
(A ∪ B)' = A' ∩ B'
(A ∩ B)' = A' ∪ B'

Generalised to n sets:
(A₁ ∪ A₂ ∪ … ∪ Aₙ)' = A₁' ∩ A₂' ∩ … ∩ Aₙ'
(A₁ ∩ A₂ ∩ … ∩ Aₙ)' = A₁' ∪ A₂' ∪ … ∪ Aₙ'

Worked example: Verify with U = {1,…,6}, A = {1,2,3}, B = {3,4,5}.
A ∪ B = {1,2,3,4,5}, so (A ∪ B)' = {6}.
A' = {4,5,6}, B' = {1,2,6}, so A' ∩ B' = {6}. ✓

Logic form: ¬(p ∨ q) ≡ ¬p ∧ ¬q — see Propositional Logic.

7) Distributive and absorption laws

Set algebra shares the same structural laws as Boolean algebra, which simplifies circuit and proof work.

LawSet formBoolean form
DistributiveA ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)A · (B + C) = A·B + A·C
DistributiveA ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)A + B·C = (A+B)·(A+C)
AbsorptionA ∪ (A ∩ B) = AA + A·B = A
AbsorptionA ∩ (A ∪ B) = AA · (A + B) = A
Worked example — simplify: (A ∩ B) ∪ (A ∩ B')
Distribute factorisation in reverse: A ∩ (B ∪ B') = A ∩ U = A.

8) Quick knowledge check

  1. If A = {a, b} and B = {b, c}, find A ∪ B and A ∩ B.
  2. With U = {1, 2, 3, 4} and A = {1, 3}, find A'.
  3. Express A − B using ∩ and complement.
  4. Apply De Morgan to simplify (A' ∪ B)'.
  5. Shade A ∩ (B ∪ C) on a three-circle Venn diagram — which regions count?
  6. Which logic gate corresponds to A ∩ B?
Show answers
  1. (1) {a, b, c}, {b}
  2. (2) {2, 4}
  3. (3) A ∩ B'
  4. (4) A ∩ B'
  5. (5) overlap of A with either B or C
  6. (6) AND.

Answers: (1) {a, b, c}, {b}; (2) {2, 4}; (3) A ∩ B'; (4) A ∩ B'; (5) overlap of A with either B or C; (6) AND.
