1) Vector space axioms

A vector space \(V\) over a field \(\mathbb{F}\) (usually \(\mathbb{R}\) or \(\mathbb{C}\)) is a set of objects called vectors, together with two operations:

  • Vector addition: \(\mathbf{u} + \mathbf{v} \in V\)
  • Scalar multiplication: \(\alpha \mathbf{v} \in V\) for \(\alpha \in \mathbb{F}\)

These must satisfy eight axioms (for all \(\mathbf{u}, \mathbf{v}, \mathbf{w} \in V\) and \(\alpha, \beta \in \mathbb{F}\)):

  • Commutativity: \(\mathbf{u} + \mathbf{v} = \mathbf{v} + \mathbf{u}\)
  • Associativity: \((\mathbf{u} + \mathbf{v}) + \mathbf{w} = \mathbf{u} + (\mathbf{v} + \mathbf{w})\)
  • Zero vector: \(\exists\,\mathbf{0}\) with \(\mathbf{v} + \mathbf{0} = \mathbf{v}\)
  • Additive inverse: \(\forall\,\mathbf{v},\;\exists\,-\mathbf{v}\) with \(\mathbf{v} + (-\mathbf{v}) = \mathbf{0}\)
  • \(\alpha(\beta \mathbf{v}) = (\alpha\beta)\mathbf{v}\)
  • \((\alpha + \beta)\mathbf{v} = \alpha\mathbf{v} + \beta\mathbf{v}\)
  • \(\alpha(\mathbf{u} + \mathbf{v}) = \alpha\mathbf{u} + \alpha\mathbf{v}\)
  • \(1 \cdot \mathbf{v} = \mathbf{v}\)
Key idea: Vectors can be added and scaled; the operations behave predictably. Geometric vectors in \(\mathbb{R}^n\) are the prototype, but functions and polynomials also form vector spaces.

Worked example: \(\mathcal{P}_2 = \{a + bx + cx^2 : a, b, c \in \mathbb{R}\}\) is a vector space. The sum of two polynomials and any scalar multiple of a polynomial remain in \(\mathcal{P}_2\).

This generalises Vectors in \(\mathbb{R}^2\) and \(\mathbb{R}^3\) to abstract settings used in Linear Transformations.

3) Subspaces

A subset \(W \subseteq V\) is a subspace if \(W\) is itself a vector space under the same operations. Equivalently, \(W\) is non-empty and closed under addition and scalar multiplication.

Subspace test: \(W \neq \emptyset\) and for all \(\mathbf{u}, \mathbf{v} \in W\), \(\alpha \in \mathbb{F}\):
  • \(\mathbf{u} + \mathbf{v} \in W\)
  • \(\alpha \mathbf{u} \in W\)
Worked example: In \(\mathbb{R}^3\), the plane \(W = \{(x,y,z) : x + y + z = 0\}\) is a subspace. Check: if \(\mathbf{u}, \mathbf{v} \in W\), then \((u_1+v_1) + (u_2+v_2) + (u_3+v_3) = 0\), so \(\mathbf{u} + \mathbf{v} \in W\). Similarly for scalar multiples.
Common mistake: A subset that fails to contain \(\mathbf{0}\) cannot be a subspace. The set \(\{(x,y) : x + y = 1\}\) in \(\mathbb{R}^2\) is not a subspace.

4) Linear combinations and span

A linear combination of vectors \(\mathbf{v}_1, \ldots, \mathbf{v}_k\) is: \[ \alpha_1 \mathbf{v}_1 + \alpha_2 \mathbf{v}_2 + \cdots + \alpha_k \mathbf{v}_k \] where \(\alpha_i \in \mathbb{F}\).

The span of a set \(S = \{\mathbf{v}_1, \ldots, \mathbf{v}_k\}\), written \(\text{span}(S)\), is the set of all linear combinations of vectors in \(S\). It is always a subspace.

Worked example: In \(\mathbb{R}^3\), \(\text{span}\{(1,0,0), (0,1,0)\}\) is the \(xy\)-plane \(\{(x,y,0) : x,y \in \mathbb{R}\}\).

We say \(S\) spans \(V\) if \(\text{span}(S) = V\) — every vector in \(V\) can be written as a linear combination of vectors in \(S\).

5) Linear independence

Vectors \(\mathbf{v}_1, \ldots, \mathbf{v}_k\) are linearly independent if the only solution to: \[ \alpha_1 \mathbf{v}_1 + \alpha_2 \mathbf{v}_2 + \cdots + \alpha_k \mathbf{v}_k = \mathbf{0} \] is \(\alpha_1 = \alpha_2 = \cdots = \alpha_k = 0\).

If a non-trivial combination gives zero, the set is linearly dependent — at least one vector is redundant and can be expressed in terms of the others.

Worked example: Are \((1,2,3)\), \((2,4,6)\) independent in \(\mathbb{R}^3\)?
Note \((2,4,6) = 2(1,2,3)\), so \(2\mathbf{v}_1 - \mathbf{v}_2 = \mathbf{0}\) with non-zero coefficients. They are dependent.
Method: Form a matrix with the vectors as columns (or rows). Linear independence holds iff the rank equals the number of vectors.

6) Basis and dimension

A basis of \(V\) is a set of vectors that is both:

  • Linearly independent
  • Spanning (spans all of \(V\))

The dimension \(\dim(V)\) is the number of vectors in any basis of \(V\). All bases of the same space have the same size — this is a fundamental theorem.

Standard basis of \(\mathbb{R}^3\): \[ \mathbf{e}_1 = \begin{pmatrix}1\\0\\0\end{pmatrix},\quad \mathbf{e}_2 = \begin{pmatrix}0\\1\\0\end{pmatrix},\quad \mathbf{e}_3 = \begin{pmatrix}0\\0\\1\end{pmatrix} \] Every \(\mathbf{v} = (a,b,c)^T = a\mathbf{e}_1 + b\mathbf{e}_2 + c\mathbf{e}_3\). So \(\dim(\mathbb{R}^3) = 3\).
Basis for \(\mathcal{P}_2\): \(\{1, x, x^2\}\). Any polynomial \(a + bx + cx^2\) is a unique linear combination. \(\dim(\mathcal{P}_2) = 3\).

7) Coordinates and the dimension theorem

Once a basis \(\mathcal{B} = \{\mathbf{v}_1, \ldots, \mathbf{v}_n\}\) is chosen, every vector \(\mathbf{v} \in V\) has unique coordinates \([\mathbf{v}]_{\mathcal{B}} = (\alpha_1, \ldots, \alpha_n)^T\) where \(\mathbf{v} = \sum \alpha_i \mathbf{v}_i\).

Dimension theorem: If \(W\) is a subspace of finite-dimensional \(V\), then \(\dim(W) \leq \dim(V)\). If \(\{\mathbf{v}_1, \ldots, \mathbf{v}_k\}\) is independent in \(V\), then \(k \leq \dim(V)\).
Worked example: Extend \(\{(1,1,0), (0,1,1)\}\) to a basis of \(\mathbb{R}^3\).
These two vectors are independent. Add \((0,0,1)\), which is not in their span. The set \(\{(1,1,0), (0,1,1), (0,0,1)\}\) is a basis of \(\mathbb{R}^3\).
Exam tip: To prove a set is a basis, verify both independence and that the count equals \(\dim(V)\), or show it spans and has the correct number of vectors.
