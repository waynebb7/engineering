Key terms & phrases

chain rule
The chain rule for differentiation states that if \(y = F(u)\) and \(u = g(x)\), then \(\dfrac{dy}{dx} = F'(u)\cdot g'(x)\).
Integration by substitution
(or \(u\)-substitution ) formalises the reverse chain rule.

1) The reverse chain rule

The chain rule for differentiation states that if \(y = F(u)\) and \(u = g(x)\), then \(\dfrac{dy}{dx} = F'(u)\cdot g'(x)\). Integration reverses this process: if we recognise an integrand of the form \(F'(g(x))\cdot g'(x)\), then

Reverse chain rule: \[ \int F'(g(x))\,g'(x)\,dx = F(g(x)) + C \]
Worked example: Evaluate \(\displaystyle\int 2x\cos(x^2)\,dx\).
Let \(u = x^2\), so \(du = 2x\,dx\). The integrand is \(\cos(u)\,du\).
\(= \sin(x^2) + C\).

This builds on Differentiation (chain rule) and Integration (antiderivatives). It is the foundation for Hyperbolic Functions integrals and Triple Integrals change-of-variables.

2) Integration by substitution

Integration by substitution (or \(u\)-substitution) formalises the reverse chain rule. To evaluate \(\displaystyle\int f(x)\,dx\):

Method:
Step 1: Choose a substitution \(u = g(x)\).
Step 2: Compute \(du = g'(x)\,dx\) and rearrange to express \(dx\) in terms of \(du\).
Step 3: Rewrite the integral entirely in terms of \(u\).
Step 4: Integrate with respect to \(u\), then substitute back.
Worked example: Evaluate \(\displaystyle\int x\sqrt{x^2 + 1}\,dx\).
Let \(u = x^2 + 1\), so \(du = 2x\,dx\) and \(x\,dx = \dfrac{du}{2}\).
\(= \dfrac{1}{2}\displaystyle\int u^{1/2}\,du = \dfrac{1}{3}(x^2+1)^{3/2} + C\).
Common mistake: Forgetting to replace \(dx\) fully. Every \(x\) in the integrand must be expressed in terms of \(u\) before integrating.

3) Recognising substitution patterns

Experienced integrators look for a function and (up to a constant factor) its derivative in the integrand. The table below lists common patterns.

Integrand pattern Substitution Result
\(f'(x)\,[F(x)]^n\) \(u = F(x)\) \(\dfrac{[F(x)]^{n+1}}{n+1} + C\)
\(\dfrac{f'(x)}{f(x)}\) \(u = f(x)\) \(\ln|f(x)| + C\)
\(f'(x)\,e^{f(x)}\) \(u = f(x)\) \(e^{f(x)} + C\)
\(f'(x)\cos(f(x))\) \(u = f(x)\) \(\sin(f(x)) + C\)
\(f'(x)\sin(f(x))\) \(u = f(x)\) \(-\cos(f(x)) + C\)
Worked example: Evaluate \(\displaystyle\int \dfrac{3x^2}{x^3 + 1}\,dx\).
Numerator is (up to factor 3) the derivative of \(x^3 + 1\).
\(= \ln|x^3 + 1| + C\).

5) Definite integrals and change of limits

For a definite integral, you may either substitute back to \(x\) before evaluating, or change the limits to \(u\)-values — both methods must agree.

Change of limits: If \(u = g(x)\), when \(x = a\) set \(u = g(a)\); when \(x = b\) set \(u = g(b)\). Then \[ \int_a^b f(x)\,dx = \int_{g(a)}^{g(b)} F(u)\,du \]
Worked example: Evaluate \(\displaystyle\int_0^1 \dfrac{2x}{x^2 + 1}\,dx\).
Let \(u = x^2 + 1\): limits change from \(x = 0 \to 1\) to \(u = 1 \to 2\).
\(= \displaystyle\int_1^2 \dfrac{du}{u} = [\ln u]_1^2 = \ln 2\).

Advantages of changing limits

Avoids back-substitution; reduces algebraic errors in timed exams.

When to keep \(x\)

If the answer must be expressed in original variables for a physical interpretation.

6) Trigonometric and hyperbolic substitutions

When integrands contain \(\sqrt{a^2 - x^2}\), \(\sqrt{a^2 + x^2}\), or \(\sqrt{x^2 - a^2}\), trigonometric or hyperbolic substitutions remove the square root.

Expression Substitution Identity used
\(\sqrt{a^2 - x^2}\) \(x = a\sin\theta\) \(1 - \sin^2\theta = \cos^2\theta\)
\(\sqrt{a^2 + x^2}\) \(x = a\sinh u\) or \(x = a\tan\theta\) \(\cosh^2 u - 1 = \sinh^2 u\)
\(\sqrt{x^2 - a^2}\) \(x = a\cosh u\) or \(x = a\sec\theta\) \(\cosh^2 u - 1 = \sinh^2 u\)
Worked example: Evaluate \(\displaystyle\int \dfrac{dx}{\sqrt{4 - x^2}}\).
Let \(x = 2\sin\theta\), \(dx = 2\cos\theta\,d\theta\).
\(= \displaystyle\int d\theta = \theta + C = \sin^{-1}\!\left(\dfrac{x}{2}\right) + C\).

See Hyperbolic Functions for logarithmic forms of the results.

7) Engineering applications

Substitution is the workhorse technique for computing work done, charge transferred, and accumulated quantities in engineering models.

Work done by a variable force

\(W = \displaystyle\int_{x_1}^{x_2} F(x)\,dx\). If \(F(x) = kx\) (spring), substitution is unnecessary; if \(F(x) = f(g(x))g'(x)\), use reverse chain rule directly.

Probability and statistics

Normalising a probability density often requires \(\displaystyle\int e^{-ax^2}\,dx\), solved via substitution and known Gaussian integrals.

Worked example: A current \(i(t) = I_0\sin(\omega t)\) flows for half a cycle. Charge transferred: \(Q = \displaystyle\int_0^{\pi/\omega} I_0\sin(\omega t)\,dt\).
Let \(u = \omega t\): \(Q = \dfrac{I_0}{\omega}[-\cos u]_0^{\pi} = \dfrac{2I_0}{\omega}\).
