1) Definition of the Laplace transform

The Laplace transform of a function \(f(t)\) (defined for \(t \geq 0\)) is: \[ \mathcal{L}\{f(t)\} = F(s) = \int_0^{\infty} f(t)\,e^{-st}\,dt \] where \(s\) is a complex variable. The transform converts differential equations in time into algebraic equations in \(s\).

Key idea: Differentiation in time becomes multiplication by \(s\) (with initial-condition terms). Integration becomes division by \(s\). This is why Laplace methods dominate circuit and control analysis.
Worked example: Unit step \(f(t) = 1\) for \(t \geq 0\): \[ F(s) = \int_0^{\infty} e^{-st}\,dt = \frac{1}{s}, \quad s > 0 \]

Prerequisite: Integration and Differentiation.

2) Common Laplace transform table

\(f(t)\), \(t \geq 0\) \(F(s) = \mathcal{L}\{f\}\)
\(1\) (unit step) \(\dfrac{1}{s}\)
\(t^n\) \(\dfrac{n!}{s^{n+1}}\)
\(e^{at}\) \(\dfrac{1}{s - a}\)
\(\sin(\omega t)\) \(\dfrac{\omega}{s^2 + \omega^2}\)
\(\cos(\omega t)\) \(\dfrac{s}{s^2 + \omega^2}\)
\(e^{at}\sin(\omega t)\) \(\dfrac{\omega}{(s-a)^2 + \omega^2}\)
\(e^{at}\cos(\omega t)\) \(\dfrac{s-a}{(s-a)^2 + \omega^2}\)
\(u(t-a)\) (delayed step) \(\dfrac{e^{-as}}{s}\)
Worked example: \(\mathcal{L}\{\cos(2t)\} = \dfrac{s}{s^2 + 4}\).
\(\mathcal{L}\{t^2\} = \dfrac{2!}{s^3} = \dfrac{2}{s^3}\).
Convention: Always state \(t \geq 0\). Functions are implicitly multiplied by the unit step unless otherwise noted.

3) Linearity and shifting theorems

The Laplace transform is linear: \[ \mathcal{L}\{\alpha f + \beta g\} = \alpha F(s) + \beta G(s) \]

First shifting theorem (frequency shift): \[ \mathcal{L}\{e^{at}f(t)\} = F(s - a) \] Example: \(\mathcal{L}\{e^{-3t}\cos(2t)\} = \dfrac{s+3}{(s+3)^2 + 4}\).
Second shifting theorem (time shift): \[ \mathcal{L}\{f(t-a)\,u(t-a)\} = e^{-as}F(s), \quad a > 0 \] A delayed signal multiplies the transform by \(e^{-as}\).
Differentiation theorem: \[ \mathcal{L}\{f'(t)\} = sF(s) - f(0), \qquad \mathcal{L}\{f''(t)\} = s^2 F(s) - sf(0) - f'(0) \]

4) Inverse Laplace transform

The inverse Laplace transform \(\mathcal{L}^{-1}\{F(s)\}\) recovers \(f(t)\) from \(F(s)\). Methods include table lookup, partial fractions, and completing the square for shifted quadratics.

Partial fractions example: Decompose \(F(s) = \dfrac{3s + 1}{s^2 + 2s + 5}\).
Complete the square: \(s^2 + 2s + 5 = (s+1)^2 + 4\).
\(F(s) = \dfrac{3(s+1) - 2}{(s+1)^2 + 4} = 3\dfrac{s+1}{(s+1)^2+4} - \dfrac{2}{2}\cdot\dfrac{2}{(s+1)^2+4}\).
\(f(t) = 3e^{-t}\cos(2t) - e^{-t}\sin(2t)\).
Common mistake: Forgetting to factor the leading coefficient when completing the square in the denominator before matching table entries.

Poles of \(F(s)\) (values where denominator is zero) correspond to natural modes — link to Eigenvalues (Electrical Engineering).

5) Solving ODEs with initial conditions

To solve \(ay'' + by' + cy = g(t)\) with initial values \(y(0)\), \(y'(0)\):

Method:
Step 1: Take \(\mathcal{L}\) of both sides, using differentiation theorems.
Step 2: Substitute initial conditions — derivatives become algebraic terms in \(Y(s)\).
Step 3: Solve for \(Y(s)\).
Step 4: Apply \(\mathcal{L}^{-1}\) via partial fractions and tables.
Worked example: \(y'' + 4y = 0\), \(y(0) = 1\), \(y'(0) = 0\).
\(s^2 Y - s + 4Y = 0 \Rightarrow Y = \dfrac{s}{s^2 + 4}\).
\(y(t) = \cos(2t)\).
RLC circuit: \(LC\,v'' + RC\,v' + v = v_s(t)\) transforms directly. Initial capacitor voltage and inductor current appear as IC terms in \(V(s)\).

6) Heaviside step function

The Heaviside unit step \(u(t)\) is: \[ u(t) = \begin{cases} 0 & t < 0 \\ 1 & t \geq 0 \end{cases} \] The shifted step \(u(t - a)\) turns on a signal at \(t = a\).

Expressing piecewise functions: A signal active only on \([a,\, b)\) can be written: \[ f(t)\,u(t-a) - f(t)\,u(t-b) \] Example: \(g(t) = 1\) for \(0 \leq t < 2\), \(0\) otherwise: \(g(t) = u(t) - u(t-2)\).
Worked example: Find \(\mathcal{L}\{u(t-1)\cdot\sin(t-1)\}\).
Let \(f(t) = \sin t\), so \(F(s) = \dfrac{1}{s^2+1}\). By time shift with \(a = 1\): \(\mathcal{L}\{u(t-1)\sin(t-1)\} = e^{-s}\dfrac{1}{s^2+1}\).
Exam tip: The function inside a time shift must be written as \(f(t-a)\), not \(f(t)\). Factor as \(f(t-a)\,u(t-a)\) before applying the second shifting theorem.

7) Engineering applications

Circuit analysis

RLC networks, transfer functions \(H(s) = V_{\text{out}}(s)/V_{\text{in}}(s)\), and impulse/step responses are standard Laplace-domain tools.

Control systems

Block diagrams, open-loop and closed-loop transfer functions, and stability (poles in left half-plane) all live in the \(s\)-domain.

Switching and delays

Power electronics, sampled systems, and delayed feedback use \(e^{-as}\) (time-shift) terms in transfer functions.

Convolution

\(\mathcal{L}\{f * g\} = F(s)G(s)\). Multiplication in \(s\) corresponds to convolution in time — the basis of filter design.

Related: Calculus (Electrical Engineering), Integration in Engineering.
