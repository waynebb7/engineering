Trig Functions

A-Level guide. Graphs of sin, cos and tan, period and amplitude, transformations, and solving trigonometric equations.

← Pure Math Subjects Catalog

1) Trigonometric functions as graphs

At A-Level, \(\sin\theta\), \(\cos\theta\), and \(\tan\theta\) are treated as functions of a real variable — usually written \(y = \sin x\), \(y = \cos x\), \(y = \tan x\) where \(x\) is in radians.

  • \(y = \sin x\): starts at \((0, 0)\), maximum 1, minimum \(-1\)
  • \(y = \cos x\): starts at \((0, 1)\), same range as sine but shifted horizontally
  • \(y = \tan x\): undefined at odd multiples of \(\frac{\pi}{2}\), range all real numbers
Key idea: The graphs are periodic — they repeat at regular intervals. Understanding period and amplitude is essential before applying transformations.

Review Unit Circle for exact values and Radian for measuring angles on the \(x\)-axis.

2) Period and amplitude

For \(y = a\sin(bx)\) or \(y = a\cos(bx)\):

\[ \text{Amplitude} = |a|, \qquad \text{Period} = \frac{2\pi}{|b|} \]

Amplitude

The amplitude is half the vertical distance between maximum and minimum. For \(y = 3\sin x\), amplitude is 3 and the graph oscillates between \(-3\) and \(3\).

Period

For \(y = \sin(2x)\), period is \(\frac{2\pi}{2} = \pi\). The graph completes one full cycle in \(\pi\) radians instead of \(2\pi\).

Worked example: State the amplitude and period of \(y = -2\cos(4x)\).
Amplitude \(= |-2| = 2\). Period \(= \frac{2\pi}{4} = \frac{\pi}{2}\). The minus sign reflects the graph in the \(x\)-axis.

3) Graphs of sin, cos and tan

Function Period Range Key features
\(y = \sin x\) \(2\pi\) \([-1, 1]\) Odd function: \(\sin(-x) = -\sin x\); zeros at \(0, \pi, 2\pi, \ldots\)
\(y = \cos x\) \(2\pi\) \([-1, 1]\) Even function: \(\cos(-x) = \cos x\); \(\cos x = \sin(x + \frac{\pi}{2})\)
\(y = \tan x\) \(\pi\) \(\mathbb{R}\) Vertical asymptotes at \(x = \frac{\pi}{2} + n\pi\); odd function
Common mistake: Stating the period of \(\tan x\) as \(2\pi\). Tangent repeats every \(\pi\) radians, not \(2\pi\).

4) Transformations

The general transformed form is:

\[ y = a\sin(bx + c) + d \quad \text{or} \quad y = a\cos(bx + c) + d \]
Parameter Effect
\(a\)Vertical stretch by \(|a|\); reflection if \(a < 0\)
\(b\)Horizontal stretch by factor \(\frac{1}{|b|}\); affects period
\(c\)Phase shift: graph shifts left by \(\frac{c}{b}\) when \(c > 0\)
\(d\)Vertical translation by \(d\)
Worked example: Describe \(y = 2\sin(3x - \pi) + 1\).
Amplitude 2, period \(\frac{2\pi}{3}\), phase shift right by \(\frac{\pi}{3}\), translated up 1 unit.
Midline: \(y = 1\). Maximum value: \(1 + 2 = 3\). Minimum value: \(1 - 2 = -1\).

5) Solving trigonometric equations

To solve equations such as \(\sin x = k\) or \(2\cos(2x) + 1 = 0\) in a given interval:

  1. Isolate the trig function (e.g. \(\cos(2x) = -\frac{1}{2}\))
  2. Find the reference angle using the unit circle or exact values
  3. Use symmetry to find all solutions in one period
  4. Add multiples of the period for the full solution set
Worked example: Solve \(\sin x = \frac{1}{2}\) for \(0 \leq x < 2\pi\).
Reference angle: \(\frac{\pi}{6}\). Sine positive in QI and QII.
Solutions: \(x = \frac{\pi}{6}\) and \(x = \pi - \frac{\pi}{6} = \frac{5\pi}{6}\).
Worked example: Solve \(2\cos(2x) + 1 = 0\) for \(0 \leq x < 2\pi\).
\(\cos(2x) = -\frac{1}{2}\). Let \(u = 2x\), solve \(\cos u = -\frac{1}{2}\) for \(0 \leq u < 4\pi\).
\(u = \frac{2\pi}{3}, \frac{4\pi}{3}, \frac{8\pi}{3}, \frac{10\pi}{3}\). So \(x = \frac{\pi}{3}, \frac{2\pi}{3}, \frac{4\pi}{3}, \frac{5\pi}{3}\).

6) Quadratic trig equations and identities

Some equations reduce to quadratics in \(\sin x\), \(\cos x\), or \(\tan x\). Use identities to rewrite first.

Worked example: Solve \(2\sin^2 x - \sin x - 1 = 0\) for \(0 \leq x < 2\pi\).
Let \(u = \sin x\): \(2u^2 - u - 1 = 0 \Rightarrow (2u + 1)(u - 1) = 0\), so \(u = -\frac{1}{2}\) or \(u = 1\).
\(\sin x = 1 \Rightarrow x = \frac{\pi}{2}\). \(\sin x = -\frac{1}{2} \Rightarrow x = \frac{7\pi}{6}, \frac{11\pi}{6}\).

Useful identities: \(\sin^2 x + \cos^2 x = 1\); \(\sin 2x = 2\sin x\cos x\); \(\cos 2x = \cos^2 x - \sin^2 x = 2\cos^2 x - 1 = 1 - 2\sin^2 x\).

7) Modelling with trig functions

Sine and cosine models describe periodic phenomena: tides, alternating current, seasonal temperature, and vibrations.

Modelling form: \(h(t) = a\sin(bt + c) + d\) or \(h(t) = a\cos(bt + c) + d\)
\(a\): amplitude (size of oscillation)
\(\frac{2\pi}{b}\): period (time for one complete cycle)
\(d\): midline (average value)

Example context

AC voltage: \(V(t) = V_0\sin(\omega t)\) where \(\omega = 2\pi f\) and \(f\) is frequency in Hz.

Graph reading

From a graph, read amplitude as half the peak-to-trough distance and period as horizontal length of one cycle.
