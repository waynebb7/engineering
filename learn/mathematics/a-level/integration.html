1) Integration as reverse differentiation

Integration is the inverse of differentiation. If \(\dfrac{dy}{dx} = f(x)\), then \(y = \int f(x)\,dx\) is the indefinite integral — the family of functions whose derivative is \(f(x)\).

Key idea: Integration finds an antiderivative. Because differentiation loses information about vertical shifts, the indefinite integral includes a constant of integration \(+C\).
Worked example: Given \(\dfrac{dy}{dx} = 3x^2\), find \(y\).
\(y = \int 3x^2\,dx = x^3 + C\).

Notation: \(\int f(x)\,dx = F(x) + C\) means \(F'(x) = f(x)\). Always include \(+C\) unless finding a particular solution with given initial conditions.

2) Standard integrals and the power rule

Reverse the differentiation results you already know:

Function Indefinite integral
\(x^n\) (\(n \neq -1\)) \(\dfrac{x^{n+1}}{n+1} + C\)
\(\dfrac{1}{x}\) \(\ln|x| + C\)
\(e^x\) \(e^x + C\)
\(\cos x\) \(\sin x + C\)
\(\sin x\) \(-\cos x + C\)
Worked example: Integrate \(\int (4x^3 - 2x + 5)\,dx\).
\(= x^4 - x^2 + 5x + C\).
Common mistake: Forgetting \(+C\) in indefinite integrals, or dividing by zero when integrating \(x^{-1}\) — use \(\ln|x|\) instead of the power rule.

3) Definite integrals

A definite integral \(\displaystyle\int_a^b f(x)\,dx\) evaluates the accumulated change of \(f(x)\) between \(x = a\) and \(x = b\). It gives a number, not a function.

Fundamental theorem of calculus: \[ \int_a^b f(x)\,dx = \big[ F(x) \big]_a^b = F(b) - F(a) \] where \(F'(x) = f(x)\).
Worked example: Evaluate \(\displaystyle\int_1^3 (2x + 1)\,dx\).
\(\displaystyle\int (2x+1)\,dx = x^2 + x\).
\(\big[x^2 + x\big]_1^3 = (9+3) - (1+1) = 10\).

Definite integrals do not need \(+C\) — any constant cancels when you subtract \(F(a)\) from \(F(b)\).

4) Area under a curve

When \(f(x) \geq 0\) on \([a, b]\), the definite integral \(\displaystyle\int_a^b f(x)\,dx\) gives the area under the curve \(y = f(x)\) between the lines \(x = a\) and \(x = b\).

Worked example: Area under \(y = x^2\) from \(x = 0\) to \(x = 2\):
\(\displaystyle\int_0^2 x^2\,dx = \big[\tfrac{x^3}{3}\big]_0^2 = \tfrac{8}{3}\) square units.
Below the \(x\)-axis: If \(f(x) < 0\) on part of the interval, the integral returns a negative value. For geometric area, split the interval and take the absolute value of each part, or integrate the top curve minus the bottom curve.

Steps

  1. Sketch the curve and shade the region
  2. Find where the curve crosses the \(x\)-axis if needed
  3. Set up \(\displaystyle\int_a^b f(x)\,dx\) with correct limits
  4. Integrate and evaluate

Units

Area is measured in square units. In applied questions, convert units if axes use different scales.

5) Area between curves

The area enclosed between two curves \(y = f(x)\) (upper) and \(y = g(x)\) (lower) from \(x = a\) to \(x = b\) is: \[ A = \int_a^b \big( f(x) - g(x) \big)\,dx \]

Worked example: Area between \(y = x\) and \(y = x^2\) from \(x = 0\) to \(x = 1\).
On \([0,1]\), \(x \geq x^2\), so upper curve is \(y = x\).
\(A = \displaystyle\int_0^1 (x - x^2)\,dx = \big[\tfrac{x^2}{2} - \tfrac{x^3}{3}\big]_0^1 = \tfrac{1}{6}\).

When curves intersect within the region, find intersection points by solving \(f(x) = g(x)\), then split the integral if the "upper" curve switches.

6) The trapezium rule

When you cannot integrate exactly, the trapezium rule approximates \(\displaystyle\int_a^b f(x)\,dx\) by splitting the interval into equal strips and approximating each strip as a trapezium.

Formula with \(n\) strips of width \(h = \dfrac{b-a}{n}\) and ordinates \(y_0, y_1, \ldots, y_n\): \[ \int_a^b f(x)\,dx \approx \frac{h}{2}\big( y_0 + 2(y_1 + y_2 + \cdots + y_{n-1}) + y_n \big) \]
Worked example: Approximate \(\displaystyle\int_0^2 x^2\,dx\) using 4 strips (\(h = 0.5\)).
\(y\)-values: \(0, 0.25, 1, 2.25, 4\).
\(\approx \dfrac{0.5}{2}(0 + 2(0.25+1+2.25) + 4) = 2.875\).
Exact value: \(\tfrac{8}{3} \approx 2.667\). More strips improve accuracy.
Exam tip: The trapezium rule gives an overestimate for concave-up curves and an underestimate for concave-down curves. State whether your answer is an over- or underestimate when asked.

7) Finding constants and interpreting integrals

A-Level questions often use definite integrals to find unknown constants or to interpret results in context.

Worked example: Given \(\displaystyle\int_1^k 2x\,dx = 8\), find \(k\) (\(k > 1\)).
\(\big[x^2\big]_1^k = k^2 - 1 = 8 \Rightarrow k^2 = 9 \Rightarrow k = 3\).

Displacement and velocity

If \(v(t)\) is velocity, \(\displaystyle\int_{t_1}^{t_2} v(t)\,dt\) gives displacement over that time interval.

Total area

For signed area (e.g. velocity below zero), split at roots and add magnitudes of each integral.

Further techniques (substitution, integration by parts) extend these ideas — see related topics in the Pure Math Subjects Catalog.

8) Quick knowledge check

  1. Find \(\displaystyle\int (6x^2 - 4)\,dx\).
  2. Evaluate \(\displaystyle\int_0^2 (3x + 1)\,dx\).
  3. Find the area under \(y = 4 - x^2\) from \(x = -2\) to \(x = 2\).
  4. Find the area between \(y = x\) and \(y = x^2\) for \(0 \leq x \leq 1\).
  5. Evaluate \(\displaystyle\int_1^4 \dfrac{1}{x}\,dx\).
  6. Use the trapezium rule with 3 strips to estimate \(\displaystyle\int_0^3 x\,dx\).
  7. Given \(\displaystyle\int_0^k x\,dx = 8\), find \(k\).
Show answers
  1. See the worked example: Given \(\dfrac{dy}{dx} = 3x^2\), find \(y\). \(y = \int 3x^2\,dx = x^3 + C\).
  2. See the worked example: Integrate \(\int (4x^3 - 2x + 5)\,dx\). \(= x^4 - x^2 + 5x + C\).
  3. See the worked example: Evaluate \(\displaystyle\int_1^3 (2x + 1)\,dx\). \(\displaystyle\int (2x+1)\,dx = x^2 + x\). \(\big[x^2 + x\big]_1^3 = (9+3) - (1+1) = 10\).
  4. See the worked example: Area under \(y = x^2\) from \(x = 0\) to \(x = 2\): \(\displaystyle\int_0^2 x^2\,dx = \big[\tfrac{x^3}{3}\big]_0^2 = \tfrac{8}{3}\) square units.
  5. See the worked example: Area between \(y = x\) and \(y = x^2\) from \(x = 0\) to \(x = 1\). On \([0,1]\), \(x \geq x^2\), so upper curve is \(y = x\). \(A = \displaystyle\int_0^1 (x - x^2)\,dx = \big[\tfrac{x^2}{2} - \tfrac{x^3}{3}\big]_0^1 = \tfrac{1}{6}\).
  6. See the worked example: Approximate \(\displaystyle\int_0^2 x^2\,dx\) using 4 strips (\(h = 0.5\)). \(y\)-values: \(0, 0.25, 1, 2.25, 4\). \(\approx \dfrac{0.5}{2}(0 + 2(0.25+1+2.25) + 4) = 2.875\). Exact value: \(\tfrac{8}{3} \approx 2.667\). More strips improve accuracy.
  7. See the worked example: Given \(\displaystyle\int_1^k 2x\,dx = 8\), find \(k\) (\(k > 1\)). \(\big[x^2\big]_1^k = k^2 - 1 = 8 \Rightarrow k^2 = 9 \Rightarrow k = 3\).

For marking: include \(+C\) for indefinite integrals, show substitution of limits for definite integrals, and sketch regions for area questions.
