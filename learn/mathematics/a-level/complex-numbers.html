1) The form \(a + bi\)

A complex number is written \(z = a + bi\), where \(a\) and \(b\) are real numbers and \(i\) is the imaginary unit defined by \(i^2 = -1\).

  • \(a = \text{Re}(z)\) is the real part
  • \(b = \text{Im}(z)\) is the imaginary part
Key idea: Complex numbers extend the real number system so that equations like \(x^2 + 1 = 0\) have solutions. They are essential in A-Level Further Mathematics and many applications in engineering and physics.
Arithmetic: Treat \(i\) like an algebraic symbol, then use \(i^2 = -1\).
\((3 + 2i) + (1 - 4i) = 4 - 2i\).

This builds on Irrational Numbers and Equations.

2) The Argand diagram

The Argand diagram represents complex numbers as points (or vectors) in a plane: the horizontal axis is the real axis and the vertical axis is the imaginary axis.

The number \(z = a + bi\) corresponds to the point \((a, b)\), or the vector from the origin to \((a, b)\).

Geometric interpretation: Adding complex numbers is vector addition in the plane. The conjugate \(z^* = a - bi\) is the reflection of \(z\) in the real axis.

Plotting

\(2 + 3i\) → point \((2, 3)\)

\(-1 + 2i\) → point \((-1, 2)\)

\(4\) (real) → point \((4, 0)\)

Pure imaginary

\(3i\) lies on the imaginary axis at \((0, 3)\).

3) Modulus and argument

For \(z = a + bi\), the modulus \(|z|\) is the distance from the origin: \[ |z| = \sqrt{a^2 + b^2} \] The argument \(\arg(z)\) is the angle \(\theta\) from the positive real axis to \(z\), usually measured anticlockwise with \(-\pi < \theta \leq \pi\).

Worked example: For \(z = 3 + 4i\):
\(|z| = \sqrt{9 + 16} = 5\).
\(\arg(z) = \tan^{-1}\!\left(\dfrac{4}{3}\right) \approx 0.927\) rad (≈ 53.1°).
Quadrants: Use \(\tan^{-1}\) carefully. For \(z = -1 + i\) (second quadrant), \(\arg(z) = \dfrac{3\pi}{4}\), not \(-\dfrac{\pi}{4}\). Sketch on the Argand diagram first.

Polar form: \(z = r(\cos\theta + i\sin\theta) = re^{i\theta}\), where \(r = |z|\) and \(\theta = \arg(z)\).

4) Complex conjugates and division

The complex conjugate of \(z = a + bi\) is \(z^* = a - bi\). Key properties:

  • \(z \cdot z^* = a^2 + b^2 = |z|^2\) (a real number)
  • \(\text{Re}(z) = \dfrac{z + z^*}{2}\), \(\text{Im}(z) = \dfrac{z - z^*}{2i}\)
Dividing complex numbers: Multiply numerator and denominator by the conjugate of the denominator.
\(\dfrac{3 + 2i}{1 - i} = \dfrac{(3+2i)(1+i)}{(1-i)(1+i)} = \dfrac{3 + 3i + 2i + 2i^2}{1 + 1} = \dfrac{1 + 5i}{2} = \dfrac{1}{2} + \dfrac{5}{2}i\).

5) Multiplying complex numbers

Multiply using algebra, then replace \(i^2\) with \(-1\). In polar form, multiplication is especially elegant: \[ r_1 e^{i\theta_1} \cdot r_2 e^{i\theta_2} = r_1 r_2 \, e^{i(\theta_1 + \theta_2)} \]

Cartesian example: \((2 + 3i)(1 - 2i)\)
\(= 2 - 4i + 3i - 6i^2 = 2 - i + 6 = 8 - i\).
Polar example: \(2\,\text{cis}\,\dfrac{\pi}{3} \times 3\,\text{cis}\,\dfrac{\pi}{6}\)
\(= 6\,\text{cis}\,\left(\dfrac{\pi}{3} + \dfrac{\pi}{6}\right) = 6\,\text{cis}\,\dfrac{\pi}{2} = 6i\).

Modulus rule

\(|z_1 z_2| = |z_1|\,|z_2|\)

Argument rule

\(\arg(z_1 z_2) = \arg(z_1) + \arg(z_2)\) (mod \(2\pi\))

6) Quadratic equations and complex roots

The quadratic formula \(x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}\) applies to all coefficients \(a, b, c \in \mathbb{R}\). When the discriminant \(\Delta = b^2 - 4ac < 0\), roots are complex conjugates.

Worked example: Solve \(x^2 + 4x + 13 = 0\).
\(\Delta = 16 - 52 = -36\).
\(x = \dfrac{-4 \pm \sqrt{-36}}{2} = \dfrac{-4 \pm 6i}{2} = -2 \pm 3i\).
Key theorem: If coefficients are real and one root is \(a + bi\), the other root is \(a - bi\). Complex roots always occur in conjugate pairs for polynomials with real coefficients.
Common mistake: Writing \(\sqrt{-36} = 6\) instead of \(6i\). Factor out \(i\): \(\sqrt{-36} = \sqrt{36}\,\sqrt{-1} = 6i\).

  4. Divide \(\dfrac{4 + 2i}{1 + i}\), giving your answer in the form \(a + bi\).
  5. Solve \(x^2 + 6x + 10 = 0\).
  6. Write \(1 + i\) in the form \(r(\cos\theta + i\sin\theta)\).
  7. Describe the locus \(|z - 2| = 3\) on the Argand diagram.

  3. 1+2 = 3.
