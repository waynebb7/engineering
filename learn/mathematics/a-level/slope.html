1) What is gradient?

The gradient (or slope) of a straight line measures how steep it is. It is the change in \(y\) for each unit increase in \(x\).

Formula: For two points \((x_1, y_1)\) and \((x_2, y_2)\), \[m = \frac{y_2 - y_1}{x_2 - x_1} = \frac{\text{rise}}{\text{run}}\]
  • Positive gradient — line slopes up left to right
  • Negative gradient — line slopes down left to right
  • Zero gradient — horizontal line
  • Undefined gradient — vertical line (\(x_2 = x_1\))

Builds on Cartesian Coordinates and Functions.

2) The equation \(y = mx + c\)

A non-vertical straight line can be written as \(y = mx + c\), where:

\(m\) — gradient

Steepness and direction of the line.

Example: \(y = 3x − 2\) has gradient 3.

\(c\) — y-intercept

Where the line crosses the \(y\)-axis (\(x = 0\)).

Example: \(y = 3x − 2\) crosses at \((0, −2)\).

Worked example: Find the equation of the line through \((0, 4)\) with gradient \(-2\).
\(c = 4\), \(m = −2\) → y = −2x + 4
Common mistake: Confusing gradient with y-intercept. In y = 5x + 1, the gradient is 5, not 1.

3) Finding the equation from two points

When you know two points on a line, calculate \(m\) first, then substitute one point to find \(c\).

Worked example: Line through \((1, 2)\) and \((4, 11)\).
\(m = \dfrac{11 − 2}{4 − 1} = \dfrac{9}{3} = 3\)
Using \((1, 2)\): \(2 = 3(1) + c\) → \(c = −1\)
Equation: y = 3x − 1

Alternative form

\(y − y_1 = m(x − x_1)\) avoids finding \(c\) first when a point and gradient are known.

Check

Substitute both original points into your final equation — both must satisfy it.

4) Parallel lines

Two straight lines are parallel if they have the same gradient but different y-intercepts. Parallel lines never meet.

Rule: Lines \(y = m_1 x + c_1\) and \(y = m_2 x + c_2\) are parallel iff \(m_1 = m_2\) and \(c_1 \ne c_2\).
Line A Line B Parallel?
y = 2x + 1 y = 2x − 5 Yes — same \(m = 2\)
y = 2x + 1 y = 2x + 1 Same line (coincident), not distinct parallels
y = 3x + 4 y = −3x + 4 No — different gradients

5) Perpendicular lines

Two non-vertical lines are perpendicular if their gradients multiply to \(-1\). One gradient is the negative reciprocal of the other.

Rule: If line 1 has gradient \(m\), a perpendicular line has gradient \(m_\perp = -\dfrac{1}{m}\).
Worked example: Find the gradient of a line perpendicular to \(y = \tfrac{2}{3}x + 5\).
\(m = \tfrac{2}{3}\) → \(m_\perp = -\tfrac{3}{2}\)
Common mistake: Forgetting the negative sign. The reciprocal of \(\tfrac{2}{3}\) is \(\tfrac{3}{2}\), but the perpendicular gradient is \(-\tfrac{3}{2}\).

Perpendicular gradients appear in Geometry (right angles) and Trigonometry (tangent and normal to a curve at A-Level extension).

6) Rate of change

Gradient represents a rate of change: how one quantity changes relative to another. In context, always state what the variables represent and include units.

Physics example

Distance–time graph: gradient = speed (m s\(^{-1}\)).

Velocity–time graph: gradient = acceleration (m s\(^{-2}\)).

Economics example

Cost vs items produced: gradient = marginal cost per item (£ per unit).

Constant rate: A straight-line graph means the rate of change is constant — the gradient is the same everywhere. Curved graphs (studied in Differentiation) have varying gradient.

7) Lines in other forms

Lines may appear as \(ax + by + c = 0\). Rearrange to \(y = mx + c\) to read off gradient and intercept, or use \(m = -\dfrac{a}{b}\) when \(b \ne 0\).

Example: \(4x − 2y + 6 = 0\)
Rearrange: \(−2y = −4x − 6\) → \(y = 2x + 3\). Gradient 2, y-intercept 3.
Form Gradient When useful
\(y = mx + c\) \(m\) Graphing, parallel/perpendicular problems
\(y − y_1 = m(x − x_1)\) \(m\) Known point and gradient
\(ax + by + c = 0\) \(-a/b\) General form in some exam questions

8) Quick knowledge check

  1. Find the gradient of the line through \((2, 5)\) and \((6, 13)\).
  2. Write the equation of the line with gradient \(-3\) and y-intercept 7.
  3. Find the equation of the line through \((1, 4)\) and \((3, 10)\).
  4. Is y = 4x − 1 parallel to y = 4x + 9?
  5. Find the gradient of a line perpendicular to y = 5x − 2.
  6. A taxi charges £3 plus £2 per mile. Write cost \(C\) as a function of miles \(n\). What is the gradient?
  7. Find gradient and y-intercept from 3x + 6y − 12 = 0.
Show answers
  1. Gradient m = 2 (rise 8, run 4).
  2. y = -3x + 7.
  3. y = 3x + 1.
  4. Yes — both lines have gradient m = 4, so they are parallel.
  5. Perpendicular gradient m = -0.2 (negative reciprocal of 5).
  6. C = 2n + 3; fixed charge £3, £2 per mile — gradient (rate) is 2.
  7. Rearrange to y = -0.5x + 2; gradient -0.5, y-intercept 2.

For marking: show rise/run calculation, state parallel/perpendicular reasoning, and include units in context questions.
