Linear Transformations

Undergraduate guide. Matrix representation, kernel and image, and the rank-nullity theorem.

← Pure Math Subjects Catalog

1) Definition of a linear transformation

A map \(T : V \to W\) between vector spaces over \(\mathbb{F}\) is a linear transformation (linear map) if for all \(\mathbf{u}, \mathbf{v} \in V\) and \(\alpha \in \mathbb{F}\):

  • Additivity: \(T(\mathbf{u} + \mathbf{v}) = T(\mathbf{u}) + T(\mathbf{v})\)
  • Homogeneity: \(T(\alpha \mathbf{v}) = \alpha T(\mathbf{v})\)

Equivalently: \(T(\alpha \mathbf{u} + \beta \mathbf{v}) = \alpha T(\mathbf{u}) + \beta T(\mathbf{v})\). Linear maps preserve linear combinations.

Worked example: \(T : \mathbb{R}^2 \to \mathbb{R}^2\) defined by \(T(x,y) = (2x + y,\, x - y)\) is linear. Check: \(T(\mathbf{u}+\mathbf{v}) = T(\mathbf{u}) + T(\mathbf{v})\) by direct substitution.
Non-example: \(T(x,y) = (x + 1,\, y)\) is not linear because \(T(\mathbf{0}) \neq \mathbf{0}\). Every linear map sends \(\mathbf{0}\) to \(\mathbf{0}\).

2) Matrix representation

If \(T : \mathbb{R}^n \to \mathbb{R}^m\) is linear, there exists a unique \(m \times n\) matrix \(\mathbf{A}\) such that \(T(\mathbf{x}) = \mathbf{A}\mathbf{x}\) for all \(\mathbf{x} \in \mathbb{R}^n\).

Column \(j\) of \(\mathbf{A}\) is \(T(\mathbf{e}_j)\) — the image of the \(j\)th standard basis vector.

Worked example: For \(T(x,y) = (2x + y,\, x - y)\): \[ T(\mathbf{e}_1) = T(1,0) = (2,1),\quad T(\mathbf{e}_2) = T(0,1) = (1,-1) \] So \(\mathbf{A} = \begin{pmatrix} 2 & 1 \\ 1 & -1 \end{pmatrix}\).

More generally, if \(\mathcal{B}\) and \(\mathcal{C}\) are bases of \(V\) and \(W\), the matrix \([\mathbf{A}]_{\mathcal{C} \leftarrow \mathcal{B}}\) represents \(T\) in those coordinates.

3) Composition and the standard matrix

If \(S : \mathbb{R}^n \to \mathbb{R}^m\) and \(T : \mathbb{R}^m \to \mathbb{R}^p\) are linear with matrices \(\mathbf{B}\) and \(\mathbf{A}\) respectively, then \(T \circ S\) has matrix \(\mathbf{A}\mathbf{B}\): \[ (T \circ S)(\mathbf{x}) = \mathbf{A}(\mathbf{B}\mathbf{x}) = (\mathbf{A}\mathbf{B})\mathbf{x} \]

Worked example: Rotation by \(90°\) in \(\mathbb{R}^2\): \[ R = \begin{pmatrix} 0 & -1 \\ 1 & 0 \end{pmatrix}, \quad R^2 = \begin{pmatrix} -1 & 0 \\ 0 & -1 \end{pmatrix} \] Two rotations give a half-turn — matrix multiplication encodes composition.

Matrix multiplication order matters: \(\mathbf{A}\mathbf{B}\) means apply \(\mathbf{B}\) first, then \(\mathbf{A}\).

4) Kernel (null space)

The kernel (or null space) of \(T : V \to W\) is: \[ \ker(T) = \{\mathbf{v} \in V : T(\mathbf{v}) = \mathbf{0}\} \] \(\ker(T)\) is a subspace of \(V\). It consists of all vectors mapped to zero — the directions collapsed by \(T\).

Finding \(\ker(T)\): For matrix \(\mathbf{A}\), solve \(\mathbf{A}\mathbf{x} = \mathbf{0}\) by row reduction. The solution space is \(\ker(T)\).
Worked example: For \(\mathbf{A} = \begin{pmatrix} 1 & 2 & 3 \\ 2 & 4 & 6 \end{pmatrix}\), row 2 is twice row 1. The kernel is \(\{(x,y,z) : x + 2y + 3z = 0\}\), a plane in \(\mathbb{R}^3\), so \(\dim(\ker T) = 2\).

\(T\) is injective (one-to-one) if and only if \(\ker(T) = \{\mathbf{0}\}\).

5) Image (range / column space)

The image (or range) of \(T : V \to W\) is: \[ \text{Im}(T) = \{T(\mathbf{v}) : \mathbf{v} \in V\} = \{ \mathbf{w} \in W : \mathbf{w} = T(\mathbf{v}) \text{ for some } \mathbf{v} \} \] \(\text{Im}(T)\) is a subspace of \(W\). For \(\mathbf{A} : \mathbb{R}^n \to \mathbb{R}^m\), the image is the column space of \(\mathbf{A}\), spanned by the columns of \(\mathbf{A}\).

Worked example: For \(\mathbf{A} = \begin{pmatrix} 1 & 0 \\ 0 & 0 \\ 1 & 0 \end{pmatrix}\), columns are \((1,0,1)^T\) and \((0,0,0)^T\). So \(\text{Im}(T) = \text{span}\{(1,0,1)^T\}\), a line in \(\mathbb{R}^3\), and \(\dim(\text{Im}\, T) = 1\).

\(T\) is surjective (onto) if \(\text{Im}(T) = W\).

6) Rank-nullity theorem

For a linear map \(T : V \to W\) with \(V\) finite-dimensional: \[ \dim(\ker T) + \dim(\text{Im}\, T) = \dim(V) \] For an \(m \times n\) matrix \(\mathbf{A}\): \(\text{nullity}(\mathbf{A}) + \text{rank}(\mathbf{A}) = n\), where \(\text{rank}(\mathbf{A}) = \dim(\text{Im}\, T)\) and \(\text{nullity}(\mathbf{A}) = \dim(\ker T)\).

Worked example: For the \(2 \times 3\) matrix above with \(\dim(\ker T) = 2\): \[ \text{rank}(\mathbf{A}) = 3 - 2 = 1 \] Only one column is linearly independent, confirming \(\dim(\text{Im}\, T) = 1\).
Consequences:
  • \(T\) injective \(\Leftrightarrow\) \(\dim(\ker T) = 0\) \(\Leftrightarrow\) rank \(= \dim(V)\)
  • \(T\) surjective (when \(\dim(V) = \dim(W)\)) \(\Leftrightarrow\) rank \(= \dim(W)\)
  • Bijective linear maps require equal dimensions and full rank

7) Isomorphism and invertibility

A linear bijection \(T : V \to W\) is an isomorphism; then \(V\) and \(W\) have the same structure. If \(\dim(V) = \dim(W) = n\), the following are equivalent for \(T\):

  • \(T\) is bijective
  • \(T\) is injective
  • \(T\) is surjective
  • Matrix \(\mathbf{A}\) has rank \(n\) (full rank, invertible when square)
Worked example: Is \(\mathbf{A} = \begin{pmatrix} 1 & 2 \\ 3 & 4 \end{pmatrix}\) invertible?
\(\det(\mathbf{A}) = 1 \cdot 4 - 2 \cdot 3 = -2 \neq 0\), so rank \(= 2 = n\). \(T\) is an isomorphism of \(\mathbb{R}^2\) onto itself.
Common mistake: Confusing rank (dimension of image) with the number of rows. Rank never exceeds \(\min(m, n)\) for an \(m \times n\) matrix.
