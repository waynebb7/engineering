1) What is a logarithm?

A logarithm is the inverse of exponentiation. If \(a^x = b\) (with \(a > 0\), \(a \ne 1\), \(b > 0\)), then \(\log_a b = x\).

Definition: \(\log_a b = x \iff a^x = b\)
Read: "log to base \(a\) of \(b\) equals \(x\)" means "\(a\) raised to \(x\) gives \(b\)".
  • \(\log_{10} 1000 = 3\) because \(10^3 = 1000\)
  • \(\log_2 32 = 5\) because \(2^5 = 32\)
  • \(\log_5 1 = 0\) because \(5^0 = 1\)

Logarithms extend Powers and connect to exponential growth in science and finance.

2) Log laws

These laws let you simplify and solve logarithmic expressions — they mirror index laws from powers.

Law Formula Condition
Product \(\log_a(xy) = \log_a x + \log_a y\) \(x, y > 0\)
Quotient \(\log_a\!\left(\dfrac{x}{y}\right) = \log_a x − \log_a y\) \(x, y > 0\)
Power \(\log_a(x^n) = n\,\log_a x\) \(x > 0\)
Change of base \(\log_a x = \dfrac{\log_b x}{\log_b a}\) \(a, b > 0\), \(a, b \ne 1\)
Common mistake: \(\log(x + y) \ne \log x + \log y\). The product law applies to multiplication, not addition.

3) Natural logarithms — ln

The natural logarithm uses base \(e \approx 2.71828\), written \(\ln x = \log_e x\). The number \(e\) arises naturally in compound growth and calculus.

Key values

\(\ln 1 = 0\)

\(\ln e = 1\)

\(\ln(e^x) = x\)

Calculator

Use the ln button for natural logs; log often means \(\log_{10}\).

Worked example: Simplify \(\ln 6 − \ln 2\).
\(= \ln\!\left(\dfrac{6}{2}\right) = \ln 3\).

\(\ln x\) is central in Differentiation and Integration of exponential functions.

4) Solving exponential equations

To solve \(a^x = b\), take logs of both sides (any base works; ln is standard at A-Level).

Worked example: Solve \(3^x = 20\).
Take ln: \(\ln(3^x) = \ln 20\)
\(x \ln 3 = \ln 20\)
\(x = \dfrac{\ln 20}{\ln 3} \approx 2.73\)
Same base: Solve \(4^{2x−1} = 16\).
Write \(16 = 4^2\): \(4^{2x−1} = 4^2\) → \(2x − 1 = 2\) → \(x = \tfrac{3}{2}\).

Strategy

  1. Rewrite with the same base if possible
  2. Otherwise take logs of both sides
  3. Use power law to bring down the exponent
  4. Solve for \(x\)

Quadratic in disguise

\(2^{2x} − 5 \cdot 2^x + 4 = 0\). Let \(u = 2^x\): \(u^2 − 5u + 4 = 0\).

5) Solving logarithmic equations

Combine log laws to a single log on each side, or convert to exponential form. Always check solutions lie in the domain (\(x > 0\) inside \(\ln x\)).

Worked example: Solve \(\log_2(x + 3) = 4\).
Exponential form: \(x + 3 = 2^4 = 16\) → \(x = 13\).
Worked example: Solve \(\ln x + \ln(x − 2) = \ln 3\).
\(\ln[x(x−2)] = \ln 3\) → \(x(x−2) = 3\) → \(x^2 − 2x − 3 = 0\) → \((x−3)(x+1) = 0\).
Reject \(x = −1\) (not in domain). Solution: \(x = 3\).
Common mistake: Forgetting to reject invalid solutions. \(\ln x\) requires \(x > 0\).

6) Graphs of logarithms and exponentials

\(y = a^x\) and \(y = \log_a x\) are reflections in the line \(y = x\). Both pass through \((0, 1)\) and \((1, 0)\) respectively for appropriate domains.

\(y = e^x\)

Always positive, increasing, passes through \((0, 1)\).

Horizontal asymptote: \(y = 0\) as \(x \to −\infty\).

\(y = \ln x\)

Defined for \(x > 0\), increasing, passes through \((1, 0)\).

Vertical asymptote: \(x = 0\).

Transformations: \(y = \ln(x − 2) + 1\) shifts right 2, up 1. Link to Functions and graph transformations.

7) Logarithms in context

Logarithmic scales compress large ranges — used in decibels (sound), pH (chemistry), and the Richter scale (earthquakes). Exponential models \(N = N_0 e^{kt}\) linearise under ln: \(\ln N = \ln N_0 + kt\).

Context Model Log form
Population growth \(P = P_0 e^{rt}\) \(\ln P = \ln P_0 + rt\)
Radioactive decay \(N = N_0 e^{−\lambda t}\) \(\ln N = \ln N_0 − \lambda t\)
Half-life \(N = \tfrac{1}{2} N_0\) Solve \(t\) using \(\ln 2\)
