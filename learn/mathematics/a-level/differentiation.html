1) The gradient function

Differentiation finds the rate of change of a function. For a curve \(y = f(x)\), the derivative \(\dfrac{dy}{dx}\) (or \(f'(x)\)) gives the gradient of the tangent at any point.

From first principles, the derivative at \(x\) is: \[ f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h} \]

Key idea: If \(\dfrac{dy}{dx} > 0\), the curve is increasing; if \(\dfrac{dy}{dx} < 0\), it is decreasing. Where \(\dfrac{dy}{dx} = 0\), the tangent is horizontal — a possible turning point.
Worked example: For \(y = x^2\), using the power rule (section 2): \(\dfrac{dy}{dx} = 2x\).
At \(x = 3\), gradient \(= 6\). At \(x = 0\), gradient \(= 0\) (minimum on the parabola).

This builds on Functions and Slope. Differentiation is the inverse process of Integration.

Worked example: Differentiate \(y = 3x^4 - 5x^2 + 7\).

Common mistake: Forgetting to subtract 1 from the power. The derivative of \(x^5\) is \(5x^4\), not \(5x^5\).

3) The chain rule

When \(y\) is a function of \(u\), and \(u\) is a function of \(x\), the chain rule gives: \[ \frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx} \]

Worked example: Differentiate \(y = (3x + 1)^5\).
Let \(u = 3x + 1\), so \(y = u^5\).
\(\dfrac{dy}{du} = 5u^4\), \(\dfrac{du}{dx} = 3\).
\(\dfrac{dy}{dx} = 5(3x+1)^4 \times 3 = 15(3x+1)^4\).
Worked example: Differentiate \(y = \sin(2x)\).
\(\dfrac{dy}{dx} = \cos(2x) \times 2 = 2\cos(2x)\).

Think "differentiate the outside, keep the inside, multiply by the derivative of the inside."

4) Product and quotient rules

When a function is a product or quotient of two simpler functions, use these rules.

Product rule

If \(y = uv\), then:

\(\dfrac{dy}{dx} = u\dfrac{dv}{dx} + v\dfrac{du}{dx}\)

Example: \(y = x^2 \sin x\)

\(\dfrac{dy}{dx} = x^2\cos x + 2x\sin x\)

Quotient rule

If \(y = \dfrac{u}{v}\), then:

\(\dfrac{dy}{dx} = \dfrac{v\dfrac{du}{dx} - u\dfrac{dv}{dx}}{v^2}\)

Example: \(y = \dfrac{x}{x+1}\)

\(\dfrac{dy}{dx} = \dfrac{(x+1)(1) - x(1)}{(x+1)^2} = \dfrac{1}{(x+1)^2}\)

Exam tip: Before using the quotient rule, check whether the function can be rewritten (e.g. as a product or using index laws) to simplify the differentiation.

5) Tangents and normals

The tangent at a point on a curve has gradient \(\dfrac{dy}{dx}\) evaluated at that point. The normal is perpendicular to the tangent.

Line equations:
Tangent at \((x_1, y_1)\): \(y - y_1 = m(x - x_1)\) where \(m = \dfrac{dy}{dx}\big|_{x=x_1}\).
Normal gradient: \(m_n = -\dfrac{1}{m}\) (when \(m \neq 0\)).
Worked example: Find the tangent to \(y = x^2\) at \((2, 4)\).
\(\dfrac{dy}{dx} = 2x\), so at \(x = 2\), gradient \(m = 4\).
Tangent: \(y - 4 = 4(x - 2)\), i.e. \(y = 4x - 4\).

Questions often give a point on the curve or ask you to find the \(x\)-coordinate where the tangent passes through a given external point.

6) Stationary points

A stationary point occurs where \(\dfrac{dy}{dx} = 0\) (or where the derivative does not exist). At such points the curve may have a local maximum, a local minimum, or a point of inflection.

Method:
Step 1: Solve \(\dfrac{dy}{dx} = 0\) to find \(x\)-coordinates.
Step 2: Substitute into \(y = f(x)\) to find \(y\)-coordinates.
Step 3: Use the second derivative or a sign table of \(\dfrac{dy}{dx}\) to classify each point.
Worked example: Find stationary points of \(y = x^3 - 3x\).
\(\dfrac{dy}{dx} = 3x^2 - 3 = 0 \Rightarrow x = \pm 1\).
Points: \((-1, 2)\) and \((1, -2)\).

Sign of \(\dfrac{dy}{dx}\)

  • + to − : local maximum
  • − to + : local minimum
  • No sign change: point of inflection

Curve sketching

Combine stationary points with intercepts and end behaviour to sketch polynomial and rational curves.

7) The second derivative

The second derivative \(\dfrac{d^2y}{dx^2}\) (or \(f''(x)\)) measures the rate of change of the gradient — how the curve is bending.

Second derivative test: At a stationary point where \(\dfrac{dy}{dx} = 0\):
If \(\dfrac{d^2y}{dx^2} > 0\) → local minimum (curve concave up).
If \(\dfrac{d^2y}{dx^2} < 0\) → local maximum (curve concave down).
If \(\dfrac{d^2y}{dx^2} = 0\) → test inconclusive; use a sign table.
Worked example: For \(y = x^3 - 3x\), \(\dfrac{d^2y}{dx^2} = 6x\).
At \(x = -1\): \(f''(-1) = -6 < 0\) → local maximum at \((-1, 2)\).
At \(x = 1\): \(f''(1) = 6 > 0\) → local minimum at \((1, -2)\).

A point of inflection is where the curve changes concavity — often where \(\dfrac{d^2y}{dx^2} = 0\), but you must confirm the concavity actually changes.

8) Quick knowledge check

  1. Differentiate \(y = 5x^3 - 2x + 4\).
  2. Differentiate \(y = (2x - 1)^4\).
  3. Differentiate \(y = x^2 e^x\).
  4. Find the gradient of \(y = \dfrac{1}{x}\) at \(x = 2\).
  5. Find the equation of the tangent to \(y = x^2 + 1\) at \((1, 2)\).
  6. Find the stationary points of \(y = x^2 - 4x + 5\).
  7. Use the second derivative to classify the stationary point of \(y = -x^2 + 6x\).
Show answers
  1. 3-2 = 1.
  2. See the worked example: Differentiate \(y = 3x^4 - 5x^2 + 7\). \(\dfrac{dy}{dx} = 12x^3 - 10x\).
  3. See the worked example: Differentiate \(y = (3x + 1)^5\). Let \(u = 3x + 1\), so \(y = u^5\). \(\dfrac{dy}{du} = 5u^4\), \(\dfrac{du}{dx} = 3\). \(\dfrac{dy}{dx} = 5(3x+1)^4 \times 3 = 15(3x+1)^4\).
  4. See the worked example: Differentiate \(y = \sin(2x)\). \(\dfrac{dy}{dx} = \cos(2x) \times 2 = 2\cos(2x)\).
  5. 2+1 = 3.
  6. 2-4 = -2.
  7. 2+6 = 8.

For marking: show each differentiation step clearly, state the gradient before writing the tangent equation, and give coordinates for all stationary points.
