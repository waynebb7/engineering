1) What is counting?

Counting is saying number names in order to find how many things there are. It is one of the first maths skills you learn, and almost every other number topic builds on it.

  • Each object is matched to one number name (one-to-one correspondence)
  • The last number you say tells you the total (the cardinal number)
  • Numbers have a fixed order: 1, 2, 3, 4, …
Key idea: Counting answers “how many?” — not “which one?” (that is ordering or labelling).

2) Counting objects reliably

To count a set of objects without mistakes:

  1. Touch or mark each object as you count it (so none is missed or counted twice)
  2. Say number names in the correct order
  3. Stop when every object has been counted — the last number is the total

Worked example

Five dots: ● ● ● ● ● → count “1, 2, 3, 4, 5” → there are 5 dots.

Common mistake

Skipping an object or counting the same one twice gives the wrong total. Count systematically (left to right, top to bottom).

3) Counting on and counting back

Once you know a starting number, you can continue without restarting at 1.

  • Counting on: start at 7 → 8, 9, 10, 11 …
  • Counting back: start at 12 → 11, 10, 9 …
Link to addition: To find 7 + 3, count on three steps from 7: 8, 9, 10 → answer 10.
Common mistake: Starting the count at 1 when you already know the starting amount. For 9 + 4, start at 9, not at 1.

4) Skip counting (multiples)

Skip counting means jumping by a fixed step instead of 1.

  • Count in 2s: 2, 4, 6, 8, 10 …
  • Count in 5s: 5, 10, 15, 20 …
  • Count in 10s: 10, 20, 30, 40 …

Skip counting prepares you for times tables, spotting patterns, and later work on multiples and factors.

KS3 note: Skip counting in 2s is the same as listing multiples of 2.

5) The number line

A number line shows numbers in order as equally spaced marks. It helps you:

  • Compare which number is greater or smaller
  • Count on or back visually
  • See gaps between numbers (useful for fractions and negatives later)

Greater and smaller

On a horizontal number line, numbers further to the right are greater; further to the left are smaller.

Example

6 is to the right of 3, so 6 > 3. We read this as “6 is greater than 3”.

6) Zero and place value

Zero (0) means “none” when counting objects. It also holds a place in larger numbers — for example, in 205 the 0 shows there are no tens.

As numbers grow past 9, we use place value:

  • Units (ones): the rightmost digit
  • Tens, hundreds, thousands — each column is worth ten times the column to its right
Example: In 3,472, the 3 means 3 thousands, the 4 means 4 hundreds, the 7 means 7 tens, and the 2 means 2 units.

7) Estimating by counting

When there are too many objects to count exactly, you can estimate:

  1. Count a small sample group (e.g. 10 objects)
  2. Estimate how many equal groups fit in the whole set
  3. Multiply to get an approximate total
Check your estimate: Estimates are useful for quick checks, but exam answers usually need an exact count unless the question says otherwise.

8) Quick knowledge check

  1. What question does counting answer?
  2. Why should you touch or mark each object when counting?
  3. Count on from 14: what are the next four numbers?
  4. Write the first five numbers when skip counting in 5s.
  5. On a number line, is 9 to the left or right of 6? Which is greater?
  6. What does the digit 0 mean in the number 408?
Show answers
  1. Counting answers “how many?” — the total number of objects.
  2. So each object is counted once (one-to-one matching) and none is missed or double-counted.
  3. 15, 16, 17, 18
  4. 5, 10, 15, 20, 25
  5. 9 is to the right of 6, so 9 > 6.
  6. 0 is a placeholder in the tens place — there are no tens in 408.

For marking: answers should mention “how many”, one-to-one matching, correct number order, and place value where relevant.
