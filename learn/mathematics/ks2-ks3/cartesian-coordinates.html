Cartesian Coordinates

KS2 to KS3 guide. How to read and plot points on a coordinate grid, understand the four quadrants, and use coordinates in graphs and shapes.

← Pure Math Subjects Catalog

1) What is a coordinate system?

The Cartesian coordinate system uses two number lines at right angles to describe the position of a point on a flat plane. It is named after René Descartes and is the foundation for graphs in maths and science.

Key idea: Every point is written as an ordered pair (x, y). The x-coordinate tells you how far left or right; the y-coordinate tells you how far up or down.

Memory aid: “Along the corridor, up the stairs” — x first (horizontal), then y (vertical).

2) The axes and origin

Horizontal axis (x-axis)

  • Positive x goes to the right
  • Negative x goes to the left
  • Labelled with numbers like a number line

Vertical axis (y-axis)

  • Positive y goes up
  • Negative y goes down
  • Crosses the x-axis at the origin
Origin: The point where the axes cross is (0, 0). It is the reference point for all other coordinates.

3) Plotting and reading points (first quadrant)

In KS2 you usually start in the first quadrant, where both x and y are zero or positive.

Worked example — plot (3, 2):
1. Start at the origin (0, 0)
2. Move 3 units right along the x-axis
3. Move 2 units up parallel to the y-axis
4. Mark the point and label it (3, 2)
Point x y Description
A 4 1 4 right, 1 up
B 0 5 On the y-axis, 5 up
C 6 0 On the x-axis, 6 right

4) Four quadrants (KS3)

When the grid extends in all directions, it is divided into four quadrants, numbered anticlockwise from the top-right.

Quadrant signs

  • Q1 (top-right): (+, +) e.g. (2, 5)
  • Q2 (top-left): (−, +) e.g. (−3, 4)
  • Q3 (bottom-left): (−, −) e.g. (−2, −6)
  • Q4 (bottom-right): (+, −) e.g. (5, −1)

Points on axes

(−4, 0) lies on the x-axis, not in a quadrant.

(0, 3) lies on the y-axis.

Link: See Negative Numbers for confidence with signs on both axes.

5) Shapes from coordinates

Joining coordinate points in order can form polygons. You can find side lengths on grid lines by counting squares.

Worked example — rectangle vertices:
(1, 1), (5, 1), (5, 4), (1, 4)
Width = 5 − 1 = 4 units. Height = 4 − 1 = 3 units. Area = 4 × 3 = 12 square units.

Translations

Move every point by the same vector.

(2, 3) translated by +3 in x, −1 in y → (5, 2)

Reflections (KS3)

Reflect in the x-axis: (x, y) → (x, −y)

Reflect in the y-axis: (x, y) → (−x, y)

Common mistake: Swapping x and y. (3, 5) and (5, 3) are different points unless the point lies on the line y = x.

6) Coordinates in graphs and real life

Graphs

  • Linear graphs: y = 2x + 1
  • Distance-time graphs in science
  • Reading values from plotted data

Real applications

  • Map grids and GPS (latitude/longitude)
  • Computer graphics and game design
  • Seating plans and chess boards
