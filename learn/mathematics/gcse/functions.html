1) What is a function?

A function is a special type of mapping: every allowed input has exactly one output. There are no one-to-many pairings.

  • Input → rule → single output
  • Written with function notation: f(x), g(x), h(x)
  • f(x) = 2x + 3 means "function f takes x and returns 2x + 3"
Key idea: If you put the same input in twice, you must get the same output both times. That is what makes a rule a function rather than a general relation.

Functions build on Variables, Equations, and Cartesian Coordinates.

2) Function notation f(x)

The notation f(x) does not mean f times x. It means "the output of function f when the input is x".

Worked example: If f(x) = 3x − 2, find f(4).
Substitute x = 4: f(4) = 3(4) − 2 = 12 − 2 = 10.

Further values

f(0) = −2

f(−1) = 3(−1) − 2 = −5

f(a) = 3a − 2 (algebraic input)

Another function

g(x) = x² + 1

g(3) = 9 + 1 = 10

g(−2) = 4 + 1 = 5

Common mistake: Writing f(4) = 3x − 2 after substituting. Once you substitute, replace x with 4 and evaluate — the answer is a number (or simplified expression), not the original formula with x still in it.

3) Function machines

A function machine shows operations applied in order: input enters, passes through one or more steps, output leaves. This is the diagram form of the same idea as f(x).

Example: Input → [×2] → [+5] → Output
Input 6 → 12 → 17. So f(6) = 17 and the rule is f(x) = 2x + 5.

Find the rule

Input 3 → output 11, input 5 → output 15.

Output increases by 2 when input increases by 1 → multiply by 2, then add 5.

Reverse (Higher)

To find input from output, work backwards through inverse operations.

Output 17 → subtract 5 → 12 → divide by 2 → input 6.

4) Functions and graphs (introduction)

A function can be shown as a graph on coordinate axes. The x-coordinate is the input; the y-coordinate is the output f(x).

  • f(x) = 2x + 1 gives a straight line (linear function)
  • Plot points from a table of values, then draw the line through them
  • The graph shows all inputs and outputs at once — not just one pair
x (input) f(x) = 2x + 1 (output) Coordinate
0 1 (0, 1)
1 3 (1, 3)
2 5 (2, 5)
3 7 (3, 7)
GCSE link: Linear graphs connect to solving Equations — the x-value where the graph crosses the x-axis satisfies f(x) = 0.

5) The vertical line test

The vertical line test decides whether a graph represents a function: if any vertical line crosses the graph more than once, it is not a function.

Passes the test ✓

Straight lines (except vertical), parabolas opening up/down, cubic curves.

Each x-value has at most one y-value.

Fails the test ✗

A circle, sideways parabola, or any shape that loops back.

One x-value would have two different y-values — one-to-many.

Remember: Vertical lines fail because they represent a single x with many y-values. Horizontal lines are fine — they represent a function that outputs the same value for every input.

6) Domain and range with functions

For a function f(x), the domain is the set of inputs you are allowed to use. The range is the set of outputs the function actually produces.

Example: f(x) = x² with domain "all real numbers".
Outputs are always ≥ 0, so the range is f(x) ≥ 0 (or "0 and all positive numbers").
Example: g(x) = 1/x
Domain: all real x except x = 0 (division by zero is undefined).
Range: all real y except y = 0.

Restrictions often involve Radicals (even roots of negatives) and Irrational Numbers in surd form at Higher tier.

7) Composite and inverse functions (Higher preview)

At GCSE Higher, you combine functions and find inverses. Foundation tier students benefit from recognising the ideas even if full manipulation is Higher-only.

Composite: fg(x)

Apply g first, then f to the result.

If f(x) = 2x and g(x) = x + 3:

fg(4) = f(g(4)) = f(7) = 14

Inverse: f⁻¹(x)

Undoes what f does — input and output swap roles.

If f(x) = 2x + 6, then f⁻¹(x) = (x − 6)/2

Check: ff⁻¹(x) = x

Higher tier: You may need to find composite rules algebraically, e.g. fg(x) = 2(x + 3) = 2x + 6, and find f⁻¹(x) by rearranging y = 2x + 6 to make x the subject.
Common mistake: Computing fg(x) as f(x) × g(x). The notation fg(x) means the composite function — apply g, then f — not multiplication.
