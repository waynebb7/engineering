1) Definition and standard examples

A metric space \((X, d)\) consists of a set \(X\) and a function \(d : X \times X \to [0, \infty)\) satisfying, for all \(x, y, z \in X\):

  • Positivity: \(d(x,y) = 0 \Leftrightarrow x = y\)
  • Symmetry: \(d(x,y) = d(y,x)\)
  • Triangle inequality: \(d(x,z) \leq d(x,y) + d(y,z)\)
Standard metrics on \(\mathbb{R}^n\):
Euclidean: \(d_2(x,y) = \|x - y\|_2\).
Taxicab: \(d_1(x,y) = \sum |x_i - y_i|\).
Supremum: \(d_\infty(x,y) = \max_i |x_i - y_i|\).
All three induce the same topology on \(\mathbb{R}^n\) (though not the same balls).

Prerequisite: Limits and Real Analysis.

2) Open sets, closed sets, and topology induced by a metric

The open ball \(B_r(x) = \{y \in X : d(x,y) < r\}\). A set \(U \subseteq X\) is open if every point has a ball contained in \(U\). A set is closed if its complement is open.

Sequential characterization: \(F\) is closed iff whenever \(x_n \in F\) and \(x_n \to x\), then \(x \in F\).
Interior, closure, boundary: \(\operatorname{int}(A)\) is the largest open set in \(A\); \(\overline{A}\) is the smallest closed set containing \(A\); \(\partial A = \overline{A} \setminus \operatorname{int}(A)\).
Concept Definition via metric
Convergence \(x_n \to x\) \(d(x_n, x) \to 0\)
Cauchy sequence \(d(x_m, x_n) \to 0\) as \(m, n \to \infty\)
Dense subset \(\overline{A} = X\)
Isolated point \(\exists\, r > 0\) with \(B_r(x) \cap A = \{x\}\)

3) Completeness and the contraction mapping theorem

\((X, d)\) is complete if every Cauchy sequence converges in \(X\). \(\mathbb{R}\) and \(\mathbb{R}^n\) are complete; \(\mathbb{Q}\) is not.

Contraction mapping theorem (Banach fixed-point theorem): If \((X,d)\) is complete and \(T : X \to X\) satisfies \(d(Tx, Ty) \leq k\,d(x,y)\) for some \(k \in [0,1)\), then \(T\) has a unique fixed point \(x^*\), and iterates \(T^n(x_0) \to x^*\).
Worked example: The map \(Tx = \cos x\) on \([0,1]\) with the usual metric is a contraction since \(|\cos' x| \leq \sin 1 < 1\). The unique fixed point is the solution to \(x = \cos x\).
Common mistake: Completeness is a property of the metric space, not of the underlying set alone. The same set can be complete or incomplete depending on the metric.

4) Compactness in metric spaces

\(K \subseteq X\) is sequentially compact if every sequence in \(K\) has a convergent subsequence with limit in \(K\). In metric spaces, sequential compactness equals total boundedness + completeness.

Heine–Borel (in \(\mathbb{R}^n\)): \(K \subseteq \mathbb{R}^n\) is compact iff \(K\) is closed and bounded.
Extreme Value Theorem: A continuous real-valued function on a compact metric space attains its maximum and minimum.

Total boundedness

For every \(\varepsilon > 0\), \(X\) can be covered by finitely many balls of radius \(\varepsilon\). Equivalent to compactness when the space is complete.

Lebesgue number lemma

For an open cover of a compact metric space, there exists \(\delta > 0\) such that every subset of diameter \(< \delta\) lies in some single cover element.

5) Connectedness and path connectedness

\(X\) is connected if it cannot be written as a disjoint union of two non-empty open sets. Path connected means any two points are joined by a continuous path.

Intermediate Value Theorem (metric form): A continuous function on a connected space takes all values between any two values in its range.
Example: \(\{(x,y) \in \mathbb{R}^2 : xy = 0\}\) (the coordinate axes) is connected but not path connected — there is no path from \((1,0)\) to \((0,1)\) staying in the set.
Convention: Path connected \(\Rightarrow\) connected, but the converse fails in general. For open subsets of \(\mathbb{R}^n\), connected implies path connected.

6) Normed spaces and uniform spaces

A normed vector space \((V, \|\cdot\|)\) induces the metric \(d(x,y) = \|x - y\|\). A Banach space is a complete normed space. Examples: \(L^p\) spaces, sequence spaces \(\ell^p\), spaces of bounded operators.

Equivalent norms: \(\|\cdot\|_a\) and \(\|\cdot\|_b\) are equivalent if \(\exists\, c, C > 0\) with \(c\|x\|_a \leq \|x\|_b \leq C\|x\|_a\). On finite-dimensional spaces, all norms are equivalent.
Uniform continuity: \(f : (X,d) \to (Y,d')\) is uniformly continuous if \(\forall\,\varepsilon > 0\,\exists\,\delta > 0\) with \(d(x,y) < \delta \Rightarrow d'(f(x), f(y)) < \varepsilon\). Continuous functions on compact domains are uniformly continuous.

See also: Banach Spaces, Hilbert Spaces, Topology.

7) Applications and research connections

Fixed-point methods

Existence proofs for ODEs, Nash equilibria, and iterative numerical algorithms rely on contraction mappings in complete metric spaces.

Fractals and Hausdorff dimension

Self-similar sets like the Cantor set and Sierpiński gasket are defined via complete metric spaces of compact sets (Hausdorff metric).

Topology bridge

Every metric induces a topology, but not every topological space is metrizable. Urysohn's metrization theorem gives conditions.

Data science

Distances (Euclidean, cosine, Wasserstein) define geometry of datasets. Metric embeddings preserve structure for visualization and learning.

Related: Real Analysis, Topology, Differential Geometry.
