1) σ-algebras and measurable spaces

A σ-algebra \(\mathcal{F}\) on a set \(X\) is a collection of subsets closed under complements and countable unions. The pair \((X, \mathcal{F})\) is a measurable space.

Definition: \(\mathcal{F} \subseteq \mathcal{P}(X)\) is a σ-algebra if:
  • \(X \in \mathcal{F}\)
  • \(A \in \mathcal{F} \Rightarrow A^c \in \mathcal{F}\)
  • \(A_1, A_2, \ldots \in \mathcal{F} \Rightarrow \bigcup_{n=1}^{\infty} A_n \in \mathcal{F}\)
The Borel σ-algebra \(\mathcal{B}(\mathbb{R}^n)\) is generated by open sets.
Example: On \(\mathbb{R}\), intervals \((a,b]\) generate the Borel σ-algebra. Every continuous function \(f: \mathbb{R} \to \mathbb{R}\) is Borel measurable.

Prerequisite: Convergence, Integration.

2) Measures and Carathéodory extension

A measure \(\mu\) on \((X, \mathcal{F})\) is a function \(\mu: \mathcal{F} \to [0, \infty]\) with \(\mu(\emptyset) = 0\) and countable additivity: \[ \mu\!\left(\bigcup_{n=1}^{\infty} A_n\right) = \sum_{n=1}^{\infty} \mu(A_n) \] for disjoint \(A_n \in \mathcal{F}\).

Carathéodory extension theorem: A σ-finite pre-measure on an algebra extends uniquely to a measure on the generated σ-algebra. This is how Lebesgue measure is constructed from length on intervals.
Key distinction: A measure space \((X, \mathcal{F}, \mu)\) carries null sets (sets of measure zero). Properties that hold almost everywhere (a.e.) ignore null sets.

3) Lebesgue measure and integration

Lebesgue measure \(\lambda\) on \(\mathbb{R}^n\) generalises length, area, and volume. A non-negative measurable function is integrated via the Lebesgue integral: \[ \int_X f\,d\mu = \sup\left\{\int_X s\,d\mu : 0 \leq s \leq f,\; s \text{ simple}\right\} \]

Simple functions: \(s = \sum_{i=1}^k a_i \mathbf{1}_{A_i}\) with \(A_i\) disjoint and measurable. For such \(s\), \(\int s\,d\mu = \sum a_i \mu(A_i)\).
Comparison with Riemann: Lebesgue integration handles limits of functions far more robustly. Every Riemann-integrable function on a bounded interval is Lebesgue integrable, but the converse fails on sets of measure zero.

4) Convergence theorems

The power of Lebesgue integration lies in interchanging limits and integrals under mild hypotheses.

Monotone Convergence Theorem (MCT): If \(0 \leq f_1 \leq f_2 \leq \cdots\) and \(f_n \to f\) pointwise, then \(\int f_n \to \int f\).
Dominated Convergence Theorem (DCT): If \(f_n \to f\) a.e. and \(|f_n| \leq g\) with \(g\) integrable, then \(\int f_n \to \int f\).
Fatou's lemma: For non-negative \(f_n\), \(\displaystyle\int \liminf f_n \leq \liminf \int f_n\).
Exam tip: DCT requires a single integrable dominating function. Without domination, \(\lim \int f_n \neq \int \lim f_n\) is possible.

5) Product measures and Fubini's theorem

Given measure spaces \((X, \mathcal{F}, \mu)\) and \((Y, \mathcal{G}, \nu)\), the product measure \(\mu \times \nu\) exists on \(X \times Y\). Fubini's theorem governs iterated integrals.

Fubini–Tonelli: If \(f: X \times Y \to \mathbb{R}\) is measurable and either \(f \geq 0\) (Tonelli) or \(\int\!\!\int |f|\,d(\mu \times \nu) < \infty\) (Fubini), then \[ \int_{X \times Y} f\,d(\mu \times \nu) = \int_X\!\left(\int_Y f(x,y)\,d\nu(y)\right)d\mu(x) \]

Product measures underpin multivariate probability, PDE theory, and Banach spaces.

6) \(L^p\) spaces and Hölder–Minkowski

For \(1 \leq p < \infty\), define \[ \|f\|_p = \left(\int_X |f|^p\,d\mu\right)^{1/p}, \qquad L^p(X, \mu) = \{f \text{ measurable}: \|f\|_p < \infty\} \] (identifying functions a.e. equal). \(L^p\) is a Banach space.

Hölder's inequality: \(\|fg\|_1 \leq \|f\|_p \|g\|_q\) where \(\frac{1}{p}+\frac{1}{q}=1\).
Minkowski's inequality: \(\|f + g\|_p \leq \|f\|_p + \|g\|_p\).
Special case: \(L^2\) is a Hilbert space with inner product \(\langle f, g \rangle = \int fg\,d\mu\). See Hilbert Spaces.

7) Radon–Nikodym theorem and applications

If \(\nu\) is σ-finite and absolutely continuous with respect to \(\mu\) (i.e. \(\mu(A)=0 \Rightarrow \nu(A)=0\)), then there exists a measurable Radon–Nikodym derivative \(f = \frac{d\nu}{d\mu}\) such that \(\nu(A) = \int_A f\,d\mu\).

Probability

Expectation, conditional expectation, and change-of-measure (Girsanov) in stochastic calculus all rely on Radon–Nikodym densities.

Functional analysis

The Riesz representation theorem for \(L^p\) duals and the Hahn–Banach theorem connect measure theory to Banach spaces.

Related: Statistics, Integration, Convergence.
