1) Pauli group and stabilisers

On \(n\) qubits, the Pauli group consists of tensor products of \(I, X, Y, Z\) with phases \(\pm 1, \pm i\). A stabiliser code is the simultaneous +1 eigenspace of an abelian subgroup \(S \subset \mathcal{P}_n\) generated by independent stabiliser operators \(g_1, \ldots, g_{n-k}\).

\[ \mathcal{C} = \{ |\psi\rangle : g_i |\psi\rangle = |\psi\rangle \text{ for all } i \} \]

Key idea: Measuring stabilisers (syndromes) detects Pauli errors because errors anticommute with some stabilisers, flipping syndrome bits.

Pauli matrices: Spin and Pauli Matrices. QEC context: Quantum Error Correction.

2) Gottesman–Knill insight

Circuits built from Clifford gates (H, S, CNOT) and stabiliser measurements can be simulated efficiently classically. Non-Clifford gates (e.g. \(T\)) are needed for universal computation — injected via magic states in fault-tolerant schemes.

Stabiliser codes are Clifford-friendly: syndrome extraction uses only Clifford operations, which is why they dominate experimental QEC roadmaps.

3) CSS codes

Calderbank–Shor–Steane (CSS) codes split \(X\)-type and \(Z\)-type stabilisers into separate commuting sets. Construct from two classical binary codes \(C_1, C_2\) with \(C_2^\perp \subseteq C_1\):

  • \(X\)-stabilisers from rows of \(C_2^\perp\)
  • \(Z\)-stabilisers from rows of \(C_1^\perp\)
Advantage: Syndrome circuits can measure all \(X\) or all \(Z\) stabilisers with simpler layouts — used in Surface Codes (CSS family).

4) Steane [[7,1,3]] code

The Steane code encodes one logical qubit in seven physical qubits with distance 3 (corrects any single Pauli error). It is the smallest CSS code achieving distance 3.

Stabilisers from the classical [7,4,3] Hamming code structure. Logical operators \(\bar{X}\) and \(\bar{Z}\) are weight-3 Pauli strings acting nontrivially on the code space while commuting with stabilisers.

Distance 3

Any single-qubit \(X,Y,Z\) error produces a unique syndrome (for most codes) and is correctable.

Transversal gates

Steane supports transversal Clifford gates — useful for fault-tolerant Clifford layers.

5) Syndrome decoding

After measuring \(m = n-k\) stabiliser bits, the syndrome \(\mathbf{s}\) maps to a most likely error \(E\) (Pauli operator) via:

  • Lookup table: feasible for small \(n\)
  • Minimum-weight perfect matching: for surface codes
  • Union-find / belief propagation: fast decoders for large lattices
Decoder latency: Real-time decoding must keep pace with syndrome generation in superconducting systems with microsecond cycles.

6) Logical operators and errors

Logical Pauli operators \(\bar{X}, \bar{Z}\) commute with all stabilisers but are not in the stabiliser group — they act nontrivially on the encoded qubit. Undetectable errors are those proportional to logical operators; code distance is the minimum weight of such an operator.

\[ \text{Correctable set: } \mathcal{E} = \{ E : \text{syndrome unique, } \mathrm{wt}(E) \leq \lfloor(d-1)/2\rfloor \} \]

7) Beyond Steane: code families

  • Surface codes: 2D nearest-neighbour — hardware friendly
  • Color codes: transversal \(T\) on some layouts (still research)
  • LDPC quantum codes: lower qubit overhead, challenging connectivity
  • Bacon–Shor, toric codes: topological stabiliser structures
Simulation: Stabiliser circuits with \(n\) qubits and polynomial gates can be simulated in \(O(n^2)\) — handy for verifying decoders and protocols.
