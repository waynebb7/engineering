1) The problem

Given a Boolean function \(f: \{0,1\}^n \to \{0,1\}\) promised to be either:

  • Constant: \(f(x) = c\) for all \(x\) (same output everywhere)
  • Balanced: \(f(x) = 0\) for exactly half of inputs, \(1\) for the other half

Classical worst case: \(2^{n-1} + 1\) queries needed. Quantum: exactly 1 oracle query.

Key idea: Deutsch–Jozsa demonstrates exponential query separation — the first proof that quantum computers can beat classical ones in a black-box setting.

Prerequisite: Quantum Circuits, Quantum Gates.

2) The Deutsch problem (\(n = 1\))

For one bit, \(f\) is either constant (\(f(0)=f(1)\)) or balanced (\(f(0) \neq f(1)\)). Classical: 2 queries. Quantum Deutsch algorithm: 1 query.

Use phase oracle \(U_f|x\rangle = (-1)^{f(x)}|x\rangle\) and ancilla prepared in \(|-\rangle\): \[ U_f|x\rangle|-\rangle = (-1)^{f(x)}|x\rangle|-\rangle \]

Circuit: \(|0\rangle \xrightarrow{H} \xrightarrow{U_f} \xrightarrow{H} \to\) measure. Outcome \(|0\rangle\) means constant; \(|1\rangle\) means balanced.

3) Phase kickback mechanism

With ancilla in \(|-\rangle = \frac{1}{\sqrt{2}}(|0\rangle - |1\rangle)\): \[ U_f|x\rangle|-\rangle = |x\rangle \frac{1}{\sqrt{2}}\bigl((-1)^{f(x)}|0\rangle - (-1)^{f(x)}|1\rangle\bigr) = (-1)^{f(x)}|x\rangle|-\rangle \]

The phase \((-1)^{f(x)}\) kicks back to the input register — this is the heart of oracle-based algorithms.

Without ancilla: Standard oracle \(|x\rangle|y\rangle \to |x\rangle|y \oplus f(x)\rangle\) also works but needs more wires; phase oracle is cleaner.

4) General Deutsch–Jozsa circuit

For \(n\) input qubits plus one ancilla:

  1. Prepare \(|0\rangle^{\otimes n}|1\rangle\)
  2. Apply \(H^{\otimes n}\) to inputs; \(H\) to ancilla \(\to |1\rangle \to |-\rangle\)
  3. Apply oracle \(U_f\)
  4. Apply \(H^{\otimes n}\) to inputs
  5. Measure all input qubits

Result: All zeros \(\Leftrightarrow\) \(f\) is constant. Any other outcome \(\Leftrightarrow\) \(f\) is balanced.

5) Why it works — interference

After step 2, the input register is uniform superposition: \[ \frac{1}{\sqrt{2^n}}\sum_{x} |x\rangle \] After the oracle: amplitudes pick up phases \((-1)^{f(x)}\).

The final \(H^{\otimes n}\) causes interference. For constant \(f\), all phases align constructively on \(|0\rangle^{\otimes n}\). For balanced \(f\), they cancel — no amplitude on all-zeros.

Not parallelism alone: Quantum advantage requires interference to extract the answer — measuring one random \(x\) would not suffice.

6) Limitations and successors

  • Promise problem: \(f\) must be constant or balanced — not useful for general functions.
  • Deterministic separation: The exponential gap holds for exact identification.
  • Simon (1994): Finds hidden period structure — precursor to Shor.
  • Bernstein–Vazirani: Finds hidden bit string in one query.

Practical algorithms: Grover, Shor.

7) Worked example: \(n = 2\)

Constant \(f(x) = 0\): oracle is identity. Final state is \(|00\rangle\) with certainty.

Balanced \(f(00)=f(11)=0\), \(f(01)=f(10)=1\): oracle applies \((-1)^{f(x)}\) phases. Final measurement never gives \(|00\rangle\).

Classical: Must query at least 3 of 4 inputs in worst case. Quantum: 1 query.
