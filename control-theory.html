Control Theory

Postgraduate guide. State-space methods, controllability, observability, Lyapunov stability, LQR, and robust \(H^\infty\) control.

← Pure Math Subjects Catalog

1) State-space representation

A linear time-invariant (LTI) system is described by \[ \dot{x}(t) = Ax(t) + Bu(t), \qquad y(t) = Cx(t) + Du(t) \] where \(x \in \mathbb{R}^n\) is the state, \(u \in \mathbb{R}^m\) the input, and \(y \in \mathbb{R}^p\) the output.

Transfer function: \(G(s) = C(sI - A)^{-1}B + D\). Poles of \(G(s)\) are eigenvalues of \(A\) (for minimal realisations) — link to Eigenvalues and Laplace Transforms.
Solution: \(x(t) = e^{At}x(0) + \int_0^t e^{A(t-\tau)}Bu(\tau)\,d\tau\). The matrix exponential \(e^{At}\) is defined via the power series \(\sum_{k=0}^{\infty} \frac{A^k t^k}{k!}\).

Prerequisite: Matrices, Differential Equations, Laplace Transforms.

2) Controllability and observability

The pair \((A, B)\) is controllable if the controllability matrix \(\mathcal{C} = [B\ AB\ A^2B\ \cdots\ A^{n-1}B]\) has full rank \(n\). The pair \((A, C)\) is observable if the observability matrix \(\mathcal{O}\) (with rows \(CA^k\)) has full rank.

Kalman decomposition: Every LTI system decomposes into controllable/uncontrollable and observable/unobservable parts. Minimal realisation requires both controllability and observability.
PBH test: \((A,B)\) is controllable iff for every eigenvalue \(\lambda\) of \(A\), \(\mathrm{rank}[\lambda I - A \mid B] = n\).
Engineering meaning: Controllability: can we steer the state anywhere? Observability: can we infer the state from output measurements?

3) Lyapunov stability

An equilibrium \(x_e\) of \(\dot{x} = f(x)\) is Lyapunov stable if trajectories starting near \(x_e\) remain near; asymptotically stable if they also converge to \(x_e\).

Lyapunov's direct method: If there exists \(V(x) > 0\) for \(x \neq 0\) with \(\dot{V}(x) \leq 0\) along trajectories, then the origin is stable. If \(\dot{V} < 0\), it is asymptotically stable.
Linear systems: \(\dot{x} = Ax\) is asymptotically stable iff all eigenvalues of \(A\) have negative real parts. For LTI systems, Lyapunov's equation \(A^T P + PA = -Q\) with \(Q > 0\) yields \(V(x) = x^T P x\).
LaSalle's invariance principle: Extends Lyapunov analysis when \(\dot{V} \leq 0\) but not strictly negative — trajectories converge to the largest invariant set where \(\dot{V} = 0\).

4) Linear Quadratic Regulator (LQR)

The LQR problem minimises the infinite-horizon cost \[ J = \int_0^{\infty} \left(x^T Q x + u^T R u\right) dt \] subject to \(\dot{x} = Ax + Bu\), with \(Q \succeq 0\) and \(R \succ 0\).

Optimal control: \(u^*(t) = -Kx(t)\) where \(K = R^{-1}B^T P\), and \(P\) solves the algebraic Riccati equation (ARE): \[ A^T P + PA - PBR^{-1}B^T P + Q = 0 \]
Properties: If \((A,B)\) is stabilisable and \((A,Q)\) is detectable, the LQR controller stabilises the closed loop and provides robustness margins (gain margin \(\geq 1/2\), phase margin \(\geq 45°\)).

Optimisation background: Optimization.

5) Robust control and \(H^\infty\) theory

Robust control addresses uncertainty in plant models. The \(H^\infty\) norm of a transfer function \(G(s)\) is \[ \|G\|_\infty = \sup_\omega \sigma_{\max}(G(j\omega)) \] the maximum singular value over frequency.

\(H^\infty\) control problem: Find a stabilising controller \(K(s)\) minimising the closed-loop sensitivity or mixed sensitivity — a min-max optimisation over worst-case disturbances.
Bounded real lemma: Connects \(\|G\|_\infty < \gamma\) to the existence of a positive-definite solution to a Riccati inequality — linking frequency-domain specs to Lyapunov/LMI methods.
Function spaces: \(H^\infty\) is a Banach algebra of bounded analytic functions in the right half-plane — see Banach Spaces and Hilbert Spaces.

6) Frequency-domain vs state-space synthesis

Classical control uses Bode and Nyquist plots; modern control uses state-space and optimisation. Both views are unified via the transfer function \(G(s) = C(sI-A)^{-1}B + D\).

Pole placement: If \((A,B)\) is controllable, feedback \(u = -Kx\) can assign all closed-loop poles (eigenvalues of \(A - BK\)) arbitrarily.
Observer design: The dual Luenberger observer \(\dot{\hat{x}} = A\hat{x} + Bu + L(y - C\hat{x})\) requires observability. Separation principle: design controller and observer independently.
Kalman filter: Optimal state estimation for stochastic systems minimises estimation error covariance — the stochastic dual of LQR.

7) Nonlinear control and Lie-theoretic methods

Feedback linearisation

Nonlinear systems \(\dot{x} = f(x) + g(x)u\) can sometimes be transformed to linear controllable form via state-dependent feedback and coordinate changes.

Lie algebras

Controllability of nonlinear systems is characterised by the Lie algebra generated by vector fields \(f\) and \(g\) — see Lie Algebras.

Model predictive control (MPC)

Receding-horizon optimisation with constraints handles multivariable plants, delays, and saturation — standard in process and aerospace industries.

Applications

Power grids, autonomous vehicles, flight control, process plants, and power electronics all rely on postgraduate-level control synthesis.

Related: Laplace Transforms, Eigenvalues (Electrical Engineering), Lie Algebras.
