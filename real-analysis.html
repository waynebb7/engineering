Real Analysis

Postgraduate guide. Rigorous limits, continuity, uniform convergence, differentiation, integration, and the bridge to measure theory.

← Pure Math Subjects Catalog

1) The real number system and completeness

Real analysis rebuilds calculus on axiomatic foundations. The reals \(\mathbb{R}\) form a complete ordered field: every non-empty subset bounded above has a least upper bound (supremum).

A metric space \((X, d)\) generalizes distance; \(\mathbb{R}\) with \(d(x,y) = |x-y|\) is the prototype. Completeness — every Cauchy sequence converges — distinguishes \(\mathbb{R}\) from \(\mathbb{Q}\).

Archimedean property: For \(x > 0\), \(\exists\, n \in \mathbb{N}\) with \(n > x\). Equivalently, \(\inf\{1/n : n \in \mathbb{N}\} = 0\).
Key idea: Completeness is the analytic engine behind the Intermediate Value Theorem, Extreme Value Theorem, and all convergence arguments on closed intervals.

Prerequisite: Limits, Metric Spaces, and Convergence.

2) Sequences, series, and uniform convergence

A sequence \((a_n)\) in \(\mathbb{R}\) converges to \(L\) if \(\forall\,\varepsilon > 0\,\exists\, N\) such that \(|a_n - L| < \varepsilon\) for \(n \geq N\). A Cauchy sequence satisfies: \(\forall\,\varepsilon > 0\,\exists\, N\) with \(|a_m - a_n| < \varepsilon\) for \(m, n \geq N\).

For functions \(f_n : E \to \mathbb{R}\), pointwise convergence means \(f_n(x) \to f(x)\) for each \(x\). Uniform convergence requires a single \(N(\varepsilon)\) working for all \(x \in E\): \[ \sup_{x \in E} |f_n(x) - f(x)| \to 0 \]

Weierstrass M-test: If \(|f_n(x)| \leq M_n\) on \(E\) and \(\sum M_n\) converges, then \(\sum f_n\) converges uniformly and absolutely on \(E\).
Common mistake: Pointwise limits of continuous functions need not be continuous. Uniform convergence on a compact set preserves continuity.

3) Continuity and uniform continuity

\(f : E \subseteq \mathbb{R} \to \mathbb{R}\) is continuous at \(a\) if \(\forall\,\varepsilon > 0\,\exists\,\delta > 0\) with \(|x - a| < \delta \Rightarrow |f(x) - f(a)| < \varepsilon\).

\(f\) is uniformly continuous on \(E\) if \(\delta\) depends only on \(\varepsilon\), not on the point \(a\). On compact sets, continuity implies uniform continuity (Heine–Cantor theorem).

Intermediate Value Theorem: If \(f\) is continuous on \([a,b]\) and \(f(a) < c < f(b)\), then \(\exists\, x \in (a,b)\) with \(f(x) = c\).
Extreme Value Theorem: A continuous function on a compact set \([a,b]\) attains its maximum and minimum.
Property Requires
IVT Connected domain (interval in \(\mathbb{R}\))
EVT Compact domain (closed and bounded in \(\mathbb{R}\))
Uniform continuity Compact domain, or Lipschitz condition globally
Interchange limit and integral Uniform convergence (or dominated convergence in Lebesgue theory)

4) Differentiation: rigorous foundations

\(f\) is differentiable at \(a\) if the limit \[ f'(a) = \lim_{h \to 0} \frac{f(a+h) - f(a)}{h} \] exists. Differentiability implies continuity, but not conversely (e.g. \(f(x) = |x|\) at 0).

Mean Value Theorem: If \(f\) is continuous on \([a,b]\) and differentiable on \((a,b)\), then \(\exists\, c \in (a,b)\) with \[ f'(c) = \frac{f(b) - f(a)}{b - a} \]
Taylor's theorem: If \(f^{(n+1)}\) exists on an interval containing \(a\), then \[ f(x) = \sum_{k=0}^{n} \frac{f^{(k)}(a)}{k!}(x-a)^k + R_n(x) \] with Lagrange remainder \(R_n(x) = \dfrac{f^{(n+1)}(\xi)}{(n+1)!}(x-a)^{n+1}\) for some \(\xi\) between \(a\) and \(x\).
Exam tip: Distinguish \(C^k\) ( \(k\) continuous derivatives) from mere \(k\)-fold differentiability. A \(C^\infty\) function need not equal its Taylor series (non-analytic smooth functions).

5) Riemann integration

For a bounded function \(f\) on \([a,b]\), the Riemann integral exists when upper and lower Darboux sums agree: \[ \int_a^b f(x)\,dx = \inf U(f,P) = \sup L(f,P) \] over partitions \(P\). Continuous functions and monotone functions are Riemann integrable.

Fundamental Theorem of Calculus: If \(F' = f\) and \(f\) is continuous, then \(\displaystyle\int_a^b f(x)\,dx = F(b) - F(a)\).
Integration by parts and substitution: Rigorous versions require absolute continuity or \(C^1\) hypotheses. Fubini's theorem for \(\displaystyle\int\!\!\int f(x,y)\,dx\,dy\) needs integrability conditions on rectangles.

Improper integrals

Integrals over unbounded domains or with singular integrands are defined as limits of proper integrals. \(\int_1^\infty x^{-p}\,dx\) converges iff \(p > 1\).

Lebesgue criterion

\(f\) is Riemann integrable iff its set of discontinuities has Lebesgue measure zero. Explains why Dirichlet's function is not Riemann integrable.

6) Measure theory and \(L^p\) spaces (preview)

Lebesgue measure \(\mu\) on \(\mathbb{R}\) extends length to a \(\sigma\)-algebra of measurable sets. The Lebesgue integral handles limits via the Dominated Convergence Theorem: if \(f_n \to f\) a.e. and \(|f_n| \leq g\) with \(g\) integrable, then \(\int f_n \to \int f\).

\(L^p\) spaces: For \(1 \leq p < \infty\), \[ L^p(\mu) = \left\{f \text{ measurable} : \int |f|^p\,d\mu < \infty\right\} / \sim \] (quotient by a.e. equality) is a Banach space. \(L^2\) is a Hilbert space with inner product \(\langle f, g \rangle = \int f\bar{g}\,d\mu\).
Key idea: Real analysis at postgraduate level is the gateway to functional analysis, PDE theory, and probability (where expectation is a Lebesgue integral).

See also: Measure Theory, Banach Spaces, Hilbert Spaces.

7) Applications and research outlook

Functional analysis

Normed spaces, operators, and spectral theory rest on completeness and measure. Banach and Hilbert spaces generalize finite-dimensional linear algebra.

PDEs and harmonic analysis

Sobolev spaces \(W^{k,p}\) measure weak differentiability. Fourier analysis on \(L^2\) uses Plancherel's theorem.

Probability theory

Rigorous probability requires measure theory: \(\mathbb{P}\) is a probability measure, random variables are measurable functions.

Signal processing

\(L^2\) convergence of Fourier series, sampling theorems, and wavelet decompositions rely on analytic estimates.

8) Quick knowledge check

  1. Prove that every Cauchy sequence in \(\mathbb{R}\) is bounded.
  2. Give an example of pointwise but not uniform convergence on \((0,1)\).
  3. State and apply the Mean Value Theorem to show \(|\sin x - \sin y| \leq |x - y|\).
  4. Prove that a uniformly continuous function on \([a,b]\) is Riemann integrable.
  5. Construct a function differentiable everywhere but with discontinuous derivative.
  6. State the Dominated Convergence Theorem and explain why it is stronger than uniform convergence.
  7. Prove \(\sqrt{2}\) is irrational using the least-upper-bound property.

For marking: provide \(\varepsilon\)-\(\delta\) proofs where requested, cite compactness or completeness explicitly, and distinguish pointwise from uniform statements.
