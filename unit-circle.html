Unit Circle

A-Level guide. Angles on the circle, definitions of sin, cos and tan, exact values, and reference angles.

← Pure Math Subjects Catalog

1) The unit circle

The unit circle is a circle centred at the origin with radius 1. Its equation is \( x^2 + y^2 = 1 \). Every point on the circle can be written as \( (\cos\theta, \sin\theta) \), where \(\theta\) is the angle measured anticlockwise from the positive \(x\)-axis.

  • At \(\theta = 0\): the point is \((1, 0)\)
  • At \(\theta = \frac{\pi}{2}\): the point is \((0, 1)\)
  • At \(\theta = \pi\): the point is \((-1, 0)\)
  • At \(\theta = \frac{3\pi}{2}\): the point is \((0, -1)\)
Key idea: Angles on the unit circle are usually measured in radians. One full turn is \(2\pi\) radians. See Radian for conversion with degrees.

This builds on Trigonometry from GCSE and connects directly to Trig Functions and circular motion in physics.

2) Defining sin, cos and tan

For an angle \(\theta\) on the unit circle, let \(P = (\cos\theta, \sin\theta)\) be the point where the terminal ray meets the circle. The trigonometric ratios are defined by the coordinates of \(P\):

\[ \cos\theta = x\text{-coordinate}, \quad \sin\theta = y\text{-coordinate}, \quad \tan\theta = \frac{\sin\theta}{\cos\theta} \quad (\cos\theta \neq 0) \]

Right-triangle link

In the first quadrant, if the opposite side is \(\sin\theta\) and adjacent is \(\cos\theta\) (both on a unit hypotenuse), then SOH CAH TOA gives the same definitions as GCSE.

All quadrants

On the unit circle, \(\sin\theta\) and \(\cos\theta\) can be negative. This extends trigonometry beyond acute angles — essential at A-Level.

Common mistake: Writing \(\tan\theta = \frac{y}{x}\) without checking \(\cos\theta \neq 0\). Tangent is undefined at \(\theta = \frac{\pi}{2}, \frac{3\pi}{2}, \ldots\)

3) Exact values

At A-Level you must know exact sine and cosine values for key angles. Derive them from special triangles or the unit circle.

\(\frac{\pi}{6}\)\(30°\)\(\frac{1}{2}\)\(\frac{\sqrt{3}}{2}\)\(\frac{1}{\sqrt{3}}\)
\(\frac{\pi}{4}\)\(45°\)\(\frac{\sqrt{2}}{2}\)\(\frac{\sqrt{2}}{2}\)\(1\)

Memory tip: For sine at \(0°, 30°, 45°, 60°, 90°\), write \(\frac{\sqrt{0}}{2}, \frac{\sqrt{1}}{2}, \frac{\sqrt{2}}{2}, \frac{\sqrt{3}}{2}, \frac{\sqrt{4}}{2}\). Cosine uses the same values in reverse order.

4) Reference angles

A reference angle \(\alpha\) is the acute angle between the terminal ray of \(\theta\) and the \(x\)-axis. It helps you find exact values in any quadrant.

Method:
Step 1: Find the reference angle \(\alpha\) (always between \(0\) and \(\frac{\pi}{2}\)).
Step 2: Use the exact value for \(\alpha\) from the table above.
Step 3: Apply the correct sign using the quadrant of \(\theta\).
Worked example: Find \(\sin\frac{5\pi}{6}\).
\(\frac{5\pi}{6}\) is in quadrant II. Reference angle: \(\alpha = \pi - \frac{5\pi}{6} = \frac{\pi}{6}\).
Sine is positive in QII, so \(\sin\frac{5\pi}{6} = \sin\frac{\pi}{6} = \frac{1}{2}\).
Worked example: Find \(\cos\frac{7\pi}{4}\).
QIV, reference angle \(\alpha = 2\pi - \frac{7\pi}{4} = \frac{\pi}{4}\). Cosine is positive in QIV.
So \(\cos\frac{7\pi}{4} = \cos\frac{\pi}{4} = \frac{\sqrt{2}}{2}\).

5) Signs in each quadrant — CAST

Only some ratios are positive in each quadrant. The CAST diagram helps you remember which:

  • Q I (0 to \(\frac{\pi}{2}\)): All positive
  • Q II (\(\frac{\pi}{2}\) to \(\pi\)): Sin positive
  • Q III (\(\pi\) to \(\frac{3\pi}{2}\)): Tan positive
  • Q IV (\(\frac{3\pi}{2}\) to \(2\pi\)): Cos positive
Example: \(\tan\frac{5\pi}{4}\) is in QIII (tan positive). Reference angle \(\frac{\pi}{4}\).
\(\tan\frac{5\pi}{4} = +\tan\frac{\pi}{4} = 1\).

Angles greater than \(2\pi\) or negative angles: add or subtract multiples of \(2\pi\) to find a coterminal angle in \([0, 2\pi)\), then apply CAST.

6) The Pythagorean identity

Since \(P = (\cos\theta, \sin\theta)\) lies on \(x^2 + y^2 = 1\), substituting gives the fundamental identity:

\[ \sin^2\theta + \cos^2\theta = 1 \]

Dividing by \(\cos^2\theta\) (when \(\cos\theta \neq 0\)) gives \(1 + \tan^2\theta = \sec^2\theta\). Dividing by \(\sin^2\theta\) gives \(1 + \cot^2\theta = \csc^2\theta\).

Worked example: Given \(\sin\theta = \frac{3}{5}\) and \(\theta\) in QII, find \(\cos\theta\).
\(\cos^2\theta = 1 - \sin^2\theta = 1 - \frac{9}{25} = \frac{16}{25}\), so \(\cos\theta = \pm\frac{4}{5}\).
Cosine is negative in QII, hence \(\cos\theta = -\frac{4}{5}\).

7) Using the unit circle in problems

The unit circle is the foundation for solving equations, sketching graphs, and working with identities.

Finding all solutions

If \(\sin\theta = \frac{1}{2}\), the reference angle is \(\frac{\pi}{6}\). Solutions in \([0, 2\pi)\) are \(\theta = \frac{\pi}{6}\) and \(\theta = \pi - \frac{\pi}{6} = \frac{5\pi}{6}\).

Parametric form

Any point on the unit circle: \(x = \cos\theta\), \(y = \sin\theta\). This links to Parametric Equations and circular motion.

Exam tip: Always state the quadrant when finding a ratio from a given ratio. A-Level questions often give \(\sin\theta\) or \(\cos\theta\) and expect the sign to be justified.
