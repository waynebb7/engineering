Hyperbolic Functions

Undergraduate guide. Definitions of \(\sinh\), \(\cosh\), and \(\tanh\), identities, derivatives, and engineering applications.

← Pure Math Subjects Catalog

1) Definitions

Hyperbolic functions are defined in analogy with trigonometric functions, but using exponentials rather than rotations on the unit circle. For any real \(x\):

Function Definition Domain / range
Hyperbolic sine \(\sinh x\) \(\dfrac{e^x - e^{-x}}{2}\) \(\mathbb{R} \to \mathbb{R}\)
Hyperbolic cosine \(\cosh x\) \(\dfrac{e^x + e^{-x}}{2}\) \(\mathbb{R} \to [1, \infty)\)
Hyperbolic tangent \(\tanh x\) \(\dfrac{\sinh x}{\cosh x} = \dfrac{e^x - e^{-x}}{e^x + e^{-x}}\) \(\mathbb{R} \to (-1, 1)\)
Key idea: Whereas \(\cos^2 x + \sin^2 x = 1\) comes from the unit circle, hyperbolic functions satisfy \(\cosh^2 x - \sinh^2 x = 1\) — the minus sign reflects the geometry of the unit hyperbola \(x^2 - y^2 = 1\).

This topic builds on Differentiation, Integration, and Trigonometric Functions.

2) Identities and symmetry

Hyperbolic functions share many identities with their trigonometric counterparts, with sign changes where appropriate. These are used constantly in second-year calculus and differential equations.

Fundamental identities

\(\cosh^2 x - \sinh^2 x = 1\)

\(1 - \tanh^2 x = \operatorname{sech}^2 x\)

\(\coth^2 x - 1 = \operatorname{csch}^2 x\)

\(\sinh(-x) = -\sinh x\) (odd)

\(\cosh(-x) = \cosh x\) (even)

Addition formulae

\(\sinh(a \pm b) = \sinh a \cosh b \pm \cosh a \sinh b\)

\(\cosh(a \pm b) = \cosh a \cosh b \pm \sinh a \sinh b\)

\(\tanh(a \pm b) = \dfrac{\tanh a \pm \tanh b}{1 \pm \tanh a \tanh b}\)

Worked example: Simplify \(\cosh^2 x + \sinh^2 x\).
Using \(\cosh^2 x - \sinh^2 x = 1\), add \(2\sinh^2 x\) to both sides:
\(\cosh^2 x + \sinh^2 x = 1 + 2\sinh^2 x = 2\cosh^2 x - 1\).
Common mistake: Writing \(\cosh^2 x + \sinh^2 x = 1\). The correct identity has a minus sign: \(\cosh^2 x - \sinh^2 x = 1\).

3) Derivatives

Differentiating the exponential definitions gives results that mirror trigonometric derivatives, but without sign changes for \(\sinh\) and \(\cosh\):

Function Derivative
\(\sinh x\) \(\cosh x\)
\(\cosh x\) \(\sinh x\)
\(\tanh x\) \(\operatorname{sech}^2 x = 1 - \tanh^2 x\)
\(\operatorname{sech} x\) \(-\operatorname{sech} x \tanh x\)
\(\operatorname{coth} x\) \(-\operatorname{csch}^2 x\)
Worked example: Differentiate \(y = \sinh(3x^2)\).
By the chain rule: \(\dfrac{dy}{dx} = \cosh(3x^2) \cdot 6x = 6x\cosh(3x^2)\).
Worked example: Differentiate \(y = \tanh x \cdot \cosh x\).
Product rule: \(\dfrac{dy}{dx} = \operatorname{sech}^2 x \cdot \cosh x + \tanh x \cdot \sinh x = \operatorname{sech} x + \sinh^2 x \operatorname{sech} x\).

4) Inverse hyperbolic functions

Because \(\cosh x\) is not one-to-one on \(\mathbb{R}\), \(\operatorname{arccosh} x\) (or \(\cosh^{-1} x\)) is defined on \(x \geq 1\) with range \([0, \infty)\). Similarly:

Inverse function Logarithmic form Derivative
\(\sinh^{-1} x\) \(\ln\!\left(x + \sqrt{x^2 + 1}\right)\) \(\dfrac{1}{\sqrt{x^2 + 1}}\)
\(\cosh^{-1} x\), \(x \geq 1\) \(\ln\!\left(x + \sqrt{x^2 - 1}\right)\) \(\dfrac{1}{\sqrt{x^2 - 1}}\)
\(\tanh^{-1} x\), \(|x| < 1\) \(\dfrac{1}{2}\ln\!\left(\dfrac{1+x}{1-x}\right)\) \(\dfrac{1}{1 - x^2}\)
Worked example: Evaluate \(\sinh^{-1}(1)\).
\(\sinh^{-1}(1) = \ln(1 + \sqrt{2}) \approx 0.881\).

Inverse hyperbolic forms appear when integrating expressions such as \(\dfrac{1}{\sqrt{x^2 \pm a^2}}\). See Inverse Chain Rule for integration techniques.

5) Integrals involving hyperbolic functions

Standard integrals follow directly from the derivative table. Many engineering integrals reduce to these forms after substitution.

Integral Result
\(\int \sinh x\,dx\) \(\cosh x + C\)
\(\int \cosh x\,dx\) \(\sinh x + C\)
\(\int \operatorname{sech}^2 x\,dx\) \(\tanh x + C\)
\(\int \dfrac{dx}{\sqrt{x^2 + a^2}}\) \(\sinh^{-1}\!\left(\dfrac{x}{a}\right) + C\)
\(\int \dfrac{dx}{\sqrt{x^2 - a^2}}\), \(x > a\) \(\cosh^{-1}\!\left(\dfrac{x}{a}\right) + C\)
Worked example: Evaluate \(\displaystyle\int_0^1 \cosh(2x)\,dx\).
\(= \left[\dfrac{\sinh(2x)}{2}\right]_0^1 = \dfrac{\sinh 2}{2} \approx 1.919\).

6) Engineering applications

Hyperbolic functions arise naturally in problems involving exponential growth and decay, symmetric boundary conditions, and hanging structures.

Catenary (cable shape)

A uniform cable under gravity hangs in the shape \(y = a\cosh(x/a)\), where \(a\) depends on tension and weight per unit length. This is the classic suspension-bridge and overhead-line problem.

Heat conduction and ODEs

Solutions to \(\dfrac{d^2y}{dx^2} - k^2 y = 0\) are linear combinations of \(\sinh(kx)\) and \(\cosh(kx)\), modelling temperature profiles in fins and steady-state diffusion.

Worked example: A cable of length \(L\) between supports at the same height satisfies \(y = a\cosh(x/a)\). The horizontal tension \(T\) and weight per unit length \(w\) are related by \(a = T/w\). Engineers use this to compute sag and maximum stress.
Exam tip: When solving ODEs with \(\pm k^2\) on the right-hand side, choose \(\sinh/\cosh\) for real distinct roots and reserve \(\sin/\cos\) for purely imaginary roots (oscillatory behaviour).

7) Osborn's rule and trigonometric analogy

Osborn's rule states: to convert a trigonometric identity to a hyperbolic one, replace \(\cos\) with \(\cosh\) and change the sign of any product (or implied product) of two sines. This helps you recall identities quickly.

Example: From \(\cos(2x) = 2\cos^2 x - 1\), Osborn's rule gives \(\cosh(2x) = 2\cosh^2 x - 1\).
From \(\sin(2x) = 2\sin x \cos x\) (a product of sines), we get \(\sinh(2x) = 2\sinh x \cosh x\) (no sign change needed here since only one sine appears).

Euler's formula links the two families: \(\cos x = \dfrac{e^{ix} + e^{-ix}}{2}\) and \(\cosh x = \dfrac{e^x + e^{-x}}{2}\). See Complex Numbers for the connection via \(e^{ix} = \cos x + i\sin x\).

8) Quick knowledge check

  1. Express \(\sinh x\) and \(\cosh x\) in terms of \(e^x\) and \(e^{-x}\).
  2. Prove that \(\cosh^2 x - \sinh^2 x = 1\).
  3. Differentiate \(y = \cosh^2(4x)\).
  4. Evaluate \(\displaystyle\int \tanh x\,dx\).
  5. Write \(\sinh^{-1}(3)\) in logarithmic form.
  6. Solve \(\dfrac{d^2y}{dx^2} - 4y = 0\) with \(y(0) = 1\), \(y'(0) = 0\).
  7. State the equation of a catenary curve and explain the physical meaning of the parameter \(a\).
Show answers
  1. See the worked example: Simplify \(\cosh^2 x + \sinh^2 x\). Using \(\cosh^2 x - \sinh^2 x = 1\), add \(2\sinh^2 x\) to both sides: \(\cosh^2 x + \sinh^2 x = 1 + 2\sinh^2 x = 2\cosh^2 x - 1\).
  2. See the worked example: Differentiate \(y = \sinh(3x^2)\). By the chain rule: \(\dfrac{dy}{dx} = \cosh(3x^2) \cdot 6x = 6x\cosh(3x^2)\).
  3. See the worked example: Differentiate \(y = \tanh x \cdot \cosh x\). Product rule: \(\dfrac{dy}{dx} = \operatorname{sech}^2 x \cdot \cosh x + \tanh x \cdot \sinh x = \operatorname{sech} x + \sinh^2 x \operatorname{sech} x\).
  4. See the worked example: Evaluate \(\sinh^{-1}(1)\). \(\sinh^{-1}(1) = \ln(1 + \sqrt{2}) \approx 0.881\).
  5. See the worked example: Evaluate \(\displaystyle\int_0^1 \cosh(2x)\,dx\). \(= \left[\dfrac{\sinh(2x)}{2}\right]_0^1 = \dfrac{\sinh 2}{2} \approx 1.919\).
  6. See the worked example: A cable of length \(L\) between supports at the same height satisfies \(y = a\cosh(x/a)\). The horizontal tension \(T\) and weight per unit length \(w\) are related by \(a = T/w\). Engineers use this to compute sag and maximum stress.
  7. See the worked example: From \(\cos(2x) = 2\cos^2 x - 1\), Osborn's rule gives \(\cosh(2x) = 2\cosh^2 x - 1\). From \(\sin(2x) = 2\sin x \cos x\) (a product of sines), we get \(\sinh(2x) = 2\sinh x \cosh x\) (no sign change needed here since only one sine appears).

For marking: show exponential definitions when proving identities, include \(+C\) for indefinite integrals, and express ODE solutions in terms of \(\sinh\) and \(\cosh\) with constants determined from initial conditions.
