Parametric Equations

A-Level guide. Parametric curves, eliminating the parameter, and calculus with parametric equations.

← Pure Math Subjects Catalog

1) What are parametric equations?

A curve is given parametrically when both coordinates are expressed in terms of a third variable, the parameter (often \(t\)): \[ x = f(t), \quad y = g(t) \]

As \(t\) varies, the point \((x, y)\) traces out a curve. This is useful when \(x\) and \(y\) are not easily related as \(y = F(x)\).

Key idea: The parameter often has a physical meaning — time, angle, or a geometric quantity — but it can be any variable that determines position on the curve.
Example: \(x = 2\cos t\), \(y = 2\sin t\) traces a circle of radius 2 centred at the origin.

This topic uses Differentiation and Trigonometry.

2) Plotting and key features

To sketch a parametric curve, choose a range of \(t\) values, calculate corresponding \((x, y)\) points, and join them in order of increasing \(t\).

Worked example: \(x = t^2\), \(y = t^3 - t\).
\(t = -2\): \((4, -6)\); \(t = -1\): \((1, 0)\); \(t = 0\): \((0, 0)\); \(t = 1\): \((1, 0)\); \(t = 2\): \((4, 6)\).
The curve crosses itself at \((1, 0)\) when \(t = \pm 1\).

Finding intercepts

  • \(x\)-intercept: set \(y = 0\), solve for \(t\), then find \(x\)
  • \(y\)-intercept: set \(x = 0\), solve for \(t\), then find \(y\)

Direction of motion

Arrows on the curve show the direction as \(t\) increases — important for interpreting motion problems.

3) Eliminating the parameter

Eliminating the parameter means finding a Cartesian equation \(y = F(x)\) or \(G(x, y) = 0\) that describes the same curve (without restricting to a particular parameter range unless stated).

Worked example: \(x = t + 1\), \(y = t^2\).
From the first equation: \(t = x - 1\).
Substitute: \(y = (x - 1)^2\). This is a parabola.
Trigonometric example: \(x = 3\cos\theta\), \(y = 3\sin\theta\).
\(\cos\theta = \dfrac{x}{3}\), \(\sin\theta = \dfrac{y}{3}\).
Using \(\cos^2\theta + \sin^2\theta = 1\): \(\dfrac{x^2}{9} + \dfrac{y^2}{9} = 1\), so \(x^2 + y^2 = 9\) (a circle).
Common mistake: After eliminating \(t\), state any restrictions. For example, \(x = t^2 \geq 0\) may limit the Cartesian equation to part of the curve.

4) Differentiating parametric curves: \(\dfrac{dy}{dx}\)

When \(x\) and \(y\) are both functions of \(t\), the gradient of the curve is: \[ \frac{dy}{dx} = \frac{dy/dt}{dx/dt} = \frac{\dot{y}}{\dot{x}} \] provided \(\dfrac{dx}{dt} \neq 0\).

Worked example: \(x = t^2\), \(y = t^3\).
\(\dfrac{dx}{dt} = 2t\), \(\dfrac{dy}{dt} = 3t^2\).
\(\dfrac{dy}{dx} = \dfrac{3t^2}{2t} = \dfrac{3t}{2}\) (for \(t \neq 0\)).

To find the gradient at a specific point, first find the value(s) of \(t\) at that point, then substitute into \(\dfrac{dy}{dx}\).

5) Tangents and normals

Once \(\dfrac{dy}{dx}\) is found in terms of \(t\), tangents and normals are found in the usual way.

Worked example: Find the tangent to \(x = t^2\), \(y = t^3\) at \(t = 2\).
Point: \((4, 8)\). Gradient: \(\dfrac{dy}{dx} = \dfrac{3(2)}{2} = 3\).
Tangent: \(y - 8 = 3(x - 4)\), i.e. \(y = 3x - 4\).
Horizontal tangent: Occurs when \(\dfrac{dy}{dt} = 0\) and \(\dfrac{dx}{dt} \neq 0\).
Vertical tangent: Occurs when \(\dfrac{dx}{dt} = 0\) and \(\dfrac{dy}{dt} \neq 0\).

For vertical tangents, the gradient is undefined in \(y = mx + c\) form — the tangent is a vertical line \(x = \text{constant}\).

6) Area under a parametric curve

The area under a parametric curve from \(t = \alpha\) to \(t = \beta\) (where the curve lies above the \(x\)-axis) is: \[ A = \int_\alpha^\beta y\,\frac{dx}{dt}\,dt \]

Worked example: Area under \(x = t^2\), \(y = t\) from \(t = 0\) to \(t = 2\).
\(A = \displaystyle\int_0^2 t \cdot 2t\,dt = \int_0^2 2t^2\,dt = \big[\tfrac{2t^3}{3}\big]_0^2 = \tfrac{16}{3}\).
Sign check: If \(\dfrac{dx}{dt} < 0\) over the interval, the integral may give a negative area — adjust limits or take the modulus as appropriate.

7) Common parametric curves

Curve Parametric form Cartesian form
Circle, radius \(a\) \(x = a\cos t\), \(y = a\sin t\) \(x^2 + y^2 = a^2\)
Ellipse \(x = a\cos t\), \(y = b\sin t\) \(\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1\)
Parabola \(x = at^2\), \(y = 2at\) \(y^2 = 4ax\)
Cycloid (intro) \(x = a(t - \sin t)\), \(y = a(1 - \cos t)\) Not a simple Cartesian form

Parametric forms often simplify calculus on curves that are awkward in Cartesian coordinates. Compare with Polar Coordinates as an alternative description.

8) Quick knowledge check

  1. Find the point on \(x = 2t\), \(y = t^2 - 3\) when \(t = -2\).
  2. Eliminate the parameter: \(x = t - 2\), \(y = t^2 + 1\).
  3. Find \(\dfrac{dy}{dx}\) for \(x = t^2 + 1\), \(y = t^3\).
  4. Find the equation of the tangent to \(x = t^2\), \(y = 2t\) at \(t = 1\).
  5. Eliminate \(\theta\) from \(x = 4\cos\theta\), \(y = 4\sin\theta\).
  6. Find the gradient of \(x = \sin t\), \(y = \cos 2t\) at \(t = \dfrac{\pi}{2}\).
  7. State the value of \(t\) where \(x = t^2 - 1\), \(y = t^3\) has a horizontal tangent.
Show answers

  4. See the worked example: \(x = t^2\), \(y = t^3\). \(\dfrac{dx}{dt} = 2t\), \(\dfrac{dy}{dt} = 3t^2\). \(\dfrac{dy}{dx} = \dfrac{3t^2}{2t} = \dfrac{3t}{2}\) (for \(t \neq 0\)).
  5. See the worked example: Find the tangent to \(x = t^2\), \(y = t^3\) at \(t = 2\). Point: \((4, 8)\). Gradient: \(\dfrac{dy}{dx} = \dfrac{3(2)}{2} = 3\). Tangent: \(y - 8 = 3(x - 4)\), i.e. \(y = 3x - 4\).
