Predicate Logic

Digital logic guide. Predicates, universal and existential quantifiers, variable scope, and translating English statements into formal logic.

← Pure Math Subjects Catalog

1) From propositions to predicates

Propositional logic treats whole statements as atoms. Predicate logic (first-order logic) analyses internal structure: subjects, properties, and relations.

  • Predicate: a property or relation that becomes a proposition when arguments are supplied — e.g. P(x): "x is prime"
  • Domain of discourse: the set of objects variables range over (e.g. integers, people, memory addresses)
  • Constants: named objects — a, 3, server_A
  • Variables: placeholders — x, y, z
Key idea: P(7) is a proposition (true). P(x) alone is not — its truth depends on x.

Builds on Propositional Logic and connects to Set Theory Basics via domains and membership.

2) Universal quantifier ∀

∀x P(x) means "for all x (in the domain), P(x) holds." It is true when every object in the domain satisfies P; false if at least one counterexample exists.

Examples:
∀x (x + 0 = x) over real numbers — true (additive identity).
∀x (x < 5) over integers — false (take x = 10).

English cues: "every", "all", "each", "for any", "no" (when combined with negation).

Common mistake: ∀ only guarantees truth for objects in the stated domain. Changing the domain can change the truth value.

3) Existential quantifier ∃

∃x P(x) means "there exists at least one x such that P(x) holds." True when you can find one witness; false when no object in the domain satisfies P.

Examples:
∃x (x² = 2) over real numbers — true (x = √2).
∃x (x² = −1) over real numbers — false; true over complex numbers.

English cues: "some", "there exists", "at least one", "for some".

Relationship: ¬∃x P(x) ≡ ∀x ¬P(x) and ¬∀x P(x) ≡ ∃x ¬P(x) — quantifier negation mirrors De Morgan at the logical level.

4) Scope and bound variables

A quantifier binds its variable within its scope — the formula immediately following it (extend with parentheses when needed).

Scope examples

∀x P(x) → Q — only P(x) is in scope of ∀

∀x (P(x) → Q) — both P(x) and Q are in scope

∃x P(x) ∧ ∃x R(x) — two separate scopes (different x allowed)

Free vs bound

  • Bound: variable governed by a quantifier in its scope
  • Free: variable not bound — often an error in closed formulas
  • Closed formula: no free variables (a genuine proposition once domain is fixed)
Parentheses matter: ∀x P(x) ∨ Q means (∀x P(x)) ∨ Q, not ∀x (P(x) ∨ Q).

5) Negation and quantifiers

Negating a quantified statement flips the quantifier and negates the predicate:

StatementNegationReading
∀x P(x)∃x ¬P(x)Not all → at least one fails
∃x P(x)∀x ¬P(x)None satisfy P
∀x (P(x) → Q(x))∃x (P(x) ∧ ¬Q(x))Someone satisfies P but not Q
Worked example: Negate "Every student passed the exam."
Formal: ∀x (S(x) → E(x)) where S(x) = student, E(x) = passed.
Negation: ∃x (S(x) ∧ ¬E(x)) — "Some student did not pass."

6) Translation from English

Translation is a core skill. Identify the domain, predicates, and whether the statement is universal or existential.

EnglishFormal logic
All engineers know logic.∀x (E(x) → L(x))
Some engineers know logic.∃x (E(x) ∧ L(x))
No signal is high and low simultaneously.∀x ¬(H(x) ∧ L(x)) or ¬∃x (H(x) ∧ L(x))
Every gate has exactly one output.∀x (G(x) → ∃!y O(x, y)) (unique ∃ written ∃!)
If a number is even, its double is even.∀x (Even(x) → Even(2x))
"All A are B" uses implication: ∀x (A(x) → B(x)). "Some A are B" uses conjunction: ∃x (A(x) ∧ B(x)). Mixing these is a frequent error.

7) Nested quantifiers

When quantifiers stack, order matters. Read nested formulas from outside in, left to right.

∀x ∃y P(x, y)

"For every x, there is some y (possibly depending on x) such that P(x,y)."

Example: every person has a mother — y can differ for each x.

∃y ∀x P(x, y)

"There is one fixed y that works for all x simultaneously."

Example: there is a universal constant c with f(x) = c for all x.

Worked example: ∀ε ∃δ P(ε, δ) (analysis flavour) — for every tolerance ε, a δ exists. Reversing to ∃δ ∀ε P(ε, δ) asserts a single δ works for all ε — a much stronger (usually false) claim.

8) Quick knowledge check

  1. Is P(x) a proposition or a predicate?
  2. Negate ∀x (x > 0) over integers.
  3. Translate: "There exists a prime number greater than 100."
  4. Does ∀x ∃y (x < y) hold over integers? Over finite sets?
  5. What is the scope of x in ∀x P(x) ∧ Q(x)?
  6. Express "Not all circuits are fault-free" using ∃.

Answers: (1) predicate; (2) ∃x (x ≤ 0); (3) ∃x (Prime(x) ∧ x > 100); (4) true over ℤ, false on finite sets; (5) only P(x) — Q(x) has free x; (6) ∃x (Circuit(x) ∧ ¬FaultFree(x)).
