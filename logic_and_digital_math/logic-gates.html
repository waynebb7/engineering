Logic Gates

Digital logic guide. AND, OR, NOT, NAND, NOR, XOR, and XNOR gates — symbols, truth tables, and why NAND alone can build any digital circuit.

← Pure Math Subjects Catalog

1) What is a logic gate?

A logic gate is an idealised electronic switch that takes one or more binary inputs (0 = LOW / false, 1 = HIGH / true) and produces a single binary output according to a fixed rule. Real chips implement these rules with transistors; mathematically they are the building blocks of Boolean algebra.

  • Inputs and outputs are always 0 or 1 — no analogue values in basic digital logic.
  • Each gate type has a standard symbol (ANSI rectangular or IEC curved shapes) and a truth table.
  • Gates combine into combinational circuits where outputs depend only on present inputs.
Key idea: Memorise the seven fundamental gates below. Every processor, memory decoder, and control unit is built from networks of them.

2) AND gate

The AND gate outputs 1 only when all inputs are 1. Boolean notation: Y = A · B or Y = A ∧ B.

ABY = A AND B
000
010
100
111

Symbol: a D-shaped body with inputs on the flat side. Think of it as a series switch — current flows only when every switch is closed.

Example: A safety interlock: motor runs (1) only when door closed AND start pressed AND no fault.

3) OR gate

The OR gate outputs 1 when at least one input is 1. Boolean notation: Y = A + B or Y = A ∨ B.

ABY = A OR B
000
011
101
111

Symbol: curved input side, pointed output. Parallel switches: any one closed allows the signal through.

Common mistake: OR is inclusive — both inputs 1 still gives 1. Exclusive OR (XOR) is a different gate.

4) NOT gate (inverter)

The NOT gate has one input and reverses it. Boolean notation: Y = A', Y = ¬A, or Y = Ā.

AY = NOT A
01
10

Symbol: a triangle (buffer) with a small circle (bubble) at the output indicating inversion.

Worked example: If A = 1, then NOT A = 0. Cascading two NOT gates returns the original value: ¬(¬A) = A.

5) NAND gate

NAND = NOT AND. Output is the inverse of AND: 0 only when all inputs are 1. Notation: Y = (A · B)' or Y = ¬(A ∧ B).

ABY = A NAND B
001
011
101
110

Symbol: AND shape with an inversion bubble on the output. NAND is functionally complete — any Boolean function can be built from NAND gates alone (see section 8).

6) NOR gate

NOR = NOT OR. Output is 1 only when all inputs are 0. Notation: Y = (A + B)' or Y = ¬(A ∨ B).

ABY = A NOR B

Like NAND, NOR is also functionally complete. Some memory technologies favour NOR-based layouts.

De Morgan link: (A · B)' = A' + B' and (A + B)' = A' · B' — see DeMorgan's Theorems for how NAND/NOR relate to AND/OR with inverters.

7) XOR and XNOR gates

XOR (exclusive OR) outputs 1 when inputs differ. XNOR (exclusive NOR) outputs 1 when inputs are equal — the inverse of XOR.

Boolean forms: A ⊕ B = A·B' + A'·B and A ⊙ B = A·B + A'·B'.

Applications: XOR implements binary addition (sum bit), parity checking, and simple encryption. XNOR is used in equality comparators and phase detectors.

8) Gate universality

A gate set is functionally complete (universal) if any Boolean function can be built using only those gates. Both NAND and NOR are universal on their own.

Build from NAND

  • NOT: A NAND A = A'
  • AND: (A NAND B) NAND (A NAND B)
  • OR: (A' NAND B')' using NAND-built NOTs

Why it matters

  • Manufacture one gate type → lower cost
  • Standard cells in ASIC libraries
  • Proof that digital logic is reducible to a single primitive
Worked example: Express XOR using only NAND:
A ⊕ B = (A NAND (A NAND B)) NAND (B NAND (A NAND B))
