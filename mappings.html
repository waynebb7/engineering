Mappings

GCSE guide. Inputs and outputs, mapping diagrams, one-to-one and many-to-one rules, and an introduction to domain and range.

← Pure Math Subjects Catalog

1) What is a mapping?

A mapping (or map) describes how each member of one set is paired with member(s) of another set. You can think of it as a rule that takes an input and produces an output.

  • Input set: often called the domain (starting values)
  • Output set: values that are actually produced — part of the range
  • The rule: e.g. "multiply by 2", "add 3", "square the number"
Key idea: Mappings appear throughout GCSE — in function machines, tables of values, graphs, and when solving Equations. They are the bridge between numbers and Functions.

2) Inputs, outputs, and mapping diagrams

A mapping diagram shows two ovals (or boxes): one for inputs, one for outputs, with arrows for each pairing.

Example — "add 2":
Inputs: 1, 2, 3 → Outputs: 3, 4, 5
Arrow from 1 to 3, from 2 to 4, from 3 to 5.

From a table

Input x: 0, 1, 2, 3

Output 2x + 1: 1, 3, 5, 7

Each input has exactly one output — a clear mapping.

From words

"The mapping takes a temperature in °C and converts to °F using a fixed formula."

One input temperature → one output temperature.

3) One-to-one and many-to-one

GCSE expects you to classify how inputs and outputs are linked. The type of mapping matters when deciding whether a rule is a function.

Type Meaning Example
One-to-one Each input maps to exactly one output; each output comes from exactly one input Multiply by 3: 2 → 6, 5 → 15
Many-to-one Different inputs can share the same output Square: 3 → 9 and −3 → 9
One-to-many One input maps to more than one output — not a function ± square root of 9 → 3 and −3 from single input 9
GCSE note: A valid function allows many-to-one but never one-to-many. Every input must have exactly one output.

4) Domain and range (introduction)

At GCSE you meet these terms informally before using them fully with graphs and function notation.

Domain

The set of allowed inputs.

Example: whole numbers 0 to 10 for a counting rule.

Example: all real x except x = 0 for 1/x.

Range

The set of possible outputs produced by the rule.

Example: for "double and add 1" with inputs 0–4, outputs are 1, 3, 5, 7, 9.

The range is the values that actually appear — not every number in the output set.

Common mistake: Confusing the codomain (all values in the output set you draw) with the range (only values that are actually hit). At GCSE, "range" usually means the outputs you get from the given domain.

5) Mappings vs general relations

A relation is any pairing between elements of two sets — it might not follow a neat rule. A mapping in GCSE usually means a clear rule applied to each input.

  • Mapping with a rule: output = 3 × input + 2 — predictable, can be written as a formula
  • General relation: a list of pairs {(1, 4), (2, 7), (3, 1)} — may have no simple formula
  • Not a mapping: one input paired with two different outputs (one-to-many)
Think of it this way: All functions are mappings, but not all relations are functions. When every input has exactly one output and there is a defined rule, you are ready for Functions and f(x) notation.

6) Describing mappings with algebra

When the rule is algebraic, you can use a letter for the input and write an expression for the output. This connects mappings to Variables and Equations.

Worked example: Mapping rule "subtract 5, then multiply by 2".
Input: n → Output: 2(n − 5) = 2n − 10
Reverse question: An output is 14. What was the input?
Solve 2n − 10 = 14 → 2n = 24 → n = 12.

Function machines in exam papers are mapping diagrams in disguise — the same input–output idea flows into composite functions at Higher tier.

7) Mapping diagrams in exam questions

Typical GCSE tasks: complete a missing value, identify the rule, or decide whether a diagram represents a function.

Finding the rule

Inputs 2 → 7, 3 → 10, 4 → 13

Outputs increase by 3 each time → "multiply input by 3, add 1" → 3x + 1

Is it a function?

Check every input oval: does it have exactly one arrow leaving it?

If any input has two arrows to different outputs, it is one-to-many — not a function.

Common mistake: Assuming a mapping is one-to-one just because it is a function. Squaring is a function (one output per input) but is many-to-one because 3 and −3 both give 9.

8) Quick knowledge check

  1. What is the output when the input is 5 for the rule "multiply by 4, then subtract 3"?
  2. Complete the mapping: input 6 → output ? for the rule 2x + 5.
  3. Is squaring a number one-to-one, many-to-one, or one-to-many?
  4. State the domain and range if inputs are {1, 2, 3} and outputs are {3, 5, 7} under the rule 2x + 1.
  5. Can a mapping be a function if two different inputs share the same output? Explain briefly.
  6. Find the input when the output is 20 for the rule 3x − 4.
