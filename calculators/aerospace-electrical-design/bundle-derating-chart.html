Bundle derating curves

AC 43.13-1B Figure 11-5, plotted from the same formula used in the Power Wire Analysis calculator.

How the factor is calculated

This page and the Power Wire Analysis calculator use a curve-fit of AC 43.13-1B Figure 11-5. The FAA chart is read graphically; this equation reproduces those five curves to about ±0.03.

Current derating factor = ((w + c) / (1 + c))−k

What the result means

The factor is a multiplier between 0.1 and 1.0. Multiply it by a wire’s free-air current rating to get the in-harness rating for that bundle condition. A factor of 1.0 means no bundle reduction; lower values mean more derating (less allowable current).

Inputs you choose

w — wires in the bundle
Count every wire in the harness bundle (1–41 on the chart). Use 1.0 with no calculation when there is only one wire (free air). For two or more wires, substitute your count as w in the formula.
Bundle loading — which curve to use
Pick the Fig 11-5 curve that matches how heavily the bundle is loaded. In AC 43.13 terms, this is the percentage of wires carrying power at or near their capacity (20%, 40%, 60%, 80%, or 100%). Higher loading → lower factor → more derating. If only a few wires are heavily loaded, use a lower loading curve (e.g. 20%). If most wires run near full current, use 100%.

Constants c and k (not user inputs)

These are fixed numbers for each loading curve. They define the shape of that curve on the chart. Select the row that matches your bundle loading percentage:

Worked example

3 wires, 100% loading: use c = 1.679 and k = 0.5143.
((3 + 1.679) / (1 + 1.679))−0.5143 ≈ 0.751 (matches the calculator default).
