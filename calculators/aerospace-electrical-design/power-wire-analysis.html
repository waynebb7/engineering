How to use this calculator

Purpose and workflow

This tool sizes aircraft power wiring by comparing every AWG column (24 through 0000) for a single circuit. For each gauge it calculates maximum run length, estimated conductor temperature T2, derated length limit L2, and voltage drop at your entered wire run — using the same chain as the ELA/PWA Excel workbook.

  1. Select wire type (resistance from the manufacturer datasheet), voltage, current, and de-rating inputs.
  2. Set wire run length and routing allowance for the voltage-drop section at the bottom of the grid.
  3. Scroll the grid horizontally and find the smallest AWG where T2 ≤ TSAFE (green) and voltage drop ≤ U (green).
  4. Check L2 against your total run if length is the limiting factor.
  5. Use Project folder & reports to connect a folder of wire workbooks, export Excel or Word reports, and restore settings from Excel.

Analysis grid (right)

Each column is one AWG size. Row labels (description and symbol) stay fixed; scroll to compare columns. A divider separates the length / temperature block from the voltage drop at run length block.

  • Green (PASS) — T2 ≤ 80% of limit, or voltage drop ≤ U.
  • Amber (CAUTION) — 80% of limit < T2 ≤ limit.
  • Red (FAIL) — T2 exceeds limit, or drop exceeds U.
  • With Apply installation temperature assessment enabled, limit = TSAFE = MIN(TR, installation limit). When disabled, limit = TR.
  • Other rows are informational (same across columns except R, L1, T2, Imax, L2, Vdrop).

Calculation sequence

For each AWG column, the calculator runs four phases in order. Symbols match the analysis grid. You are checking two things: (1) conductor temperature T2 stays at or below the installation limit TSAFE, and (2) voltage drop at your entered run length stays at or below the limit U.

Phase 1 — De-rated ampacity (can this gauge carry the load current I?)

  1. Free-air current x — ampacity in still air at your T1 and TR, from AC 43.13 Fig 11-4 (scaled from tabulated Iref at a 210 °C rise).
    $$\Delta T = T_R - T_1$$ $$I_x = I_{\mathrm{ref}} \left(\frac{\Delta T}{210}\right)^{1/2}$$

    Grid: row x; row % = 100 × I / x.

  2. Bundle factor y — reduces ampacity for wires packed together (Fig 11-5; wire count n and loading % set curve constants c and k).
    $$y = \left(\frac{n + c}{1 + c}\right)^{-k}$$

    Grid: row y (shown next to bundle controls in the parameter panel).

  3. Altitude factor z — reduces ampacity in thin air (Fig 11-6; h = altitude in thousands of feet, 0–100 kft, minimum 0.7).
    $$z = 0.7 + 0.3 \left(1 - \frac{h}{100}\right)^{1.5}$$

    Grid: row z (shown next to altitude in the parameter panel).

  4. Maximum allowable current Imax — the highest current this AWG can carry after all de-rating. If I > Imax, the wire is overloaded before temperature is even estimated.
    $$I_{\max} = x \cdot y \cdot z$$

    Grid: row IMAX; row I / IMAX = I / Imax.

Phase 2 — Conductor temperature (how hot does the wire get?)

  1. T2 factor fT2 — how much of the allowed temperature rise is used, based on how close I is to Imax. Standard is selectable (ARP4404 default; AC 43 legacy).
    $$f_{T2} = \left(\frac{I}{I_{\max}}\right)^2 \quad \text{(SAE ARP4404C §9.3.4.2 — default)}$$ $$f_{T2} = \sqrt{\frac{I}{I_{\max}}} \quad \text{(AC 43.13-1B — legacy ELA)}$$

    Grid: row I / IMAX squared (ARP4404) or √ (AC 43).

  2. Estimated conductor temperature T2 — interpolated between ambient T1 and rating TR. Pass/fail uses the active temperature limit (TSAFE when installation assessment is on, TR when off).
    $$T_2 = T_1 + (T_R - T_1) \times f_{T2}$$

    Grid: row T2.

Phase 3 — Maximum length at drop limit U (informational — not your actual run)

These rows answer: “How long could this wire be before Vdrop reaches U?” They use the allowable drop U from parameters — not the wire run you enter for voltage drop.

  1. Max length at 20 °C L1 — length limit if resistance stayed at catalog RL (Ω/ft at 20 °C) and drop equalled U.
    $$L_1 = \frac{U}{I \cdot R_L}$$

    Grid: row L1 (ft).

  2. De-rated max length L2 — same limit adjusted for copper heating to T2 (higher temperature → higher resistance → shorter allowable length).
    $$L_2 = \frac{254.5 \times L_1}{234.5 + T_2}$$

    Grid: rows L2 (in, ft, m).

  3. Resistance at T2 R2 — copper R at operating temperature (used in Phase 4).
    $$R_2 = R_L \times \frac{234.5 + T_2}{254.5}$$

    Grid: not shown separately; embedded in the Vdrop formula below.

Constants 234.5 and 254.5 — copper resistance temperature correction per ARP4404C §9.3.4.2 (234.5 + 20 °C reference → 254.5). RL is resistance per foot at 20 °C; R1000 = 1000 × RL.

Phase 4 — Voltage drop at your run (does your actual installation exceed U?)

Below the divider in the grid. Uses total run length L from Voltage drop — wire run in the parameter panel (one-way run + routing allowance).

  1. Calculated drop Vdrop — I through heated copper over your entered length L. Highlighted green when Vdrop ≤ U, red when it exceeds the limit.
    $$V_{\mathrm{drop}} = I \cdot R_L \cdot L \times \frac{234.5 + T_2}{254.5}$$

    Grid: row U (Vdrop) compared to parameter U; row L shows the total run used.

Choosing a wire size

Start at the smallest gauge that meets your physical routing constraints, then move to larger wire until T2 and Vdrop are both acceptable (green). If I > Imax for every column, the circuit is overloaded even before temperature rise — increase gauge or reduce bundle/altitude penalties. When I ≫ Imax, ARP4404 T2 rises very sharply (squared term); that is expected behaviour, not a software error.

Advanced Heat-Balance Model Physics-Based Conductor Temperature Estimation

This supplementary model estimates conductor temperature using electrical heat generation balanced against convection, radiation and conduction heat rejection. It provides a traceable physics-based engineering estimate for substantiation and sensitivity assessment, including installation-specific analysis. It does not replace the primary standards-based wire sizing calculation above or Design Authority approval.
