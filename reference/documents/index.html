Adding documents: Copy files into reference/documents/files/ (PDF, .docx, or .xlsx), register them in reference/documents/manifest.json (set a group for section headings; use a versions array to combine related editions on one tile), then run python scripts/build-documents-manifest.py. Calculators can link to registered documents from their reference panel.
