Quantum Circuits

Quantum guide. The circuit model of quantum computation — wires, gates, measurement, depth, and how algorithms are expressed diagrammatically.

← Quantum Subjects Catalog

1) What is a quantum circuit?

A quantum circuit is a sequence of unitary gates applied to qubits, followed by measurement. Each horizontal wire represents one qubit; time flows left to right.

Mathematically, an \(n\)-qubit circuit implements a unitary \(U \in U(2^n)\) on an initial state \(|0\rangle^{\otimes n}\), then measures in the computational basis.

Key idea: Circuits are the standard programming model for gate-based quantum computers (IBM, Google, IonQ, etc.).

Prerequisite: Quantum Gates, Qubits.

2) Circuit elements

  • Initialisation: \(|0\rangle\) on each wire (physical reset or cooling)
  • Single-qubit gates: \(H, X, R_z(\theta)\), etc. — boxes on one wire
  • Multi-qubit gates: CNOT, CZ — control dot + target symbol
  • Measurement: Meter symbol; classical bits store outcomes
  • Ancillas: Extra qubits initialised to \(|0\rangle\), often discarded
No fan-out: Unlike classical circuits, you cannot branch a qubit wire — the no-cloning theorem forbids it.

3) Circuit depth and width

Width = number of qubits. Depth = longest path of gates (counting sequential layers; gates on disjoint wires in the same layer run in parallel).

Circuit depth matters for NISQ devices where decoherence limits how many gates you can run before errors dominate.

Hardware constraints: Not all logical two-qubit gates are native. Compilers decompose circuits into the device's native gate set and connectivity graph.

See NISQ Era Computing.

4) Example: Bell state preparation

Circuit on two qubits:

  1. Apply \(H\) to qubit 0: \(|00\rangle \to \frac{1}{\sqrt{2}}(|00\rangle + |10\rangle)\)
  2. Apply CNOT (control 0, target 1): \(\to \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle) = |\Phi^+\rangle\)

Matrix form: \(\mathrm{CNOT}(H \otimes I)|00\rangle = |\Phi^+\rangle\).

Depth 2, width 2: One of the simplest entangling circuits — used in teleportation and Bell tests.

5) Universality and decomposition

Any unitary on \(n\) qubits decomposes into CNOT gates and single-qubit rotations. Solovay–Kitaev guarantees efficient approximation with a finite universal gate set.

Typical decomposition cost: \(O(4^n)\) CNOTs for a generic \(n\)-qubit unitary — exponential in general, but structured algorithms (QFT, oracles) have efficient circuits.

Quantum Fourier Transform: \(O(n^2)\) gates — backbone of Shor's algorithm.

6) Oracles and algorithm circuits

An oracle \(O_f\) is a black-box unitary encoding a function \(f\): \[ O_f|x\rangle|y\rangle = |x\rangle|y \oplus f(x)\rangle \] Algorithms like Deutsch–Jozsa and Grover query an oracle repeatedly.

Phase oracles implement \(|x\rangle \to (-1)^{f(x)}|x\rangle\) using one ancilla: conjugate a CNOT-style construction with \(H\) on the ancilla.

7) Mid-circuit measurement and classical control

Some circuits measure qubits mid-execution and apply gates conditioned on classical outcomes (feed-forward). Teleportation and error correction use this pattern.

Dynamic circuits (measure-then-apply) are supported on modern hardware but add latency from classical processing loops.

Deferred measurement principle: Mid-circuit measurements can be pushed to the end with equivalent unitary evolution — but not always practical on noisy hardware.
