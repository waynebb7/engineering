Quantum Error Correction

Quantum guide. Encoding logical qubits into redundant physical qubits so noise can be detected and corrected — the foundation of scalable quantum computing.

← Quantum Subjects Catalog

1) Why classical error correction fails

Classical bits can be copied and measured without disturbance. Quantum states cannot be cloned (No-Cloning Theorem) and measurement collapses superposition. Noise in quantum hardware is continuous (bit-flip, phase-flip, depolarising) and accumulates in deep circuits.

  • Bit-flip error: \(X|0\rangle = |1\rangle\) — like classical flip
  • Phase-flip error: \(Z|+\rangle = |-\rangle\) — no classical analogue
  • Decoherence: entanglement with environment — see Quantum Channels
Key idea: Quantum error correction (QEC) spreads information across many physical qubits and measures syndromes that reveal error types without collapsing the encoded logical information.

2) The three-qubit bit-flip code

Encode one logical qubit: \(|0_L\rangle = |000\rangle\), \(|1_L\rangle = |111\rangle\). A single \(X\) error on one physical qubit is detectable by parity checks \(Z_0 Z_1\) and \(Z_1 Z_2\).

Syndrome bits identify which qubit flipped; apply corrective \(X\) on that qubit. This code corrects one bit-flip but not phase errors.

Limitation: Real noise includes \(Z\) and correlated errors — need richer codes like Stabilizer Codes and Surface Codes.

3) Shor's nine-qubit code

Peter Shor's code was the first to protect against arbitrary single-qubit errors (bit and phase). It concatenates bit-flip and phase-flip protection with 9 physical qubits per logical qubit.

\[ |0_L\rangle \propto (|000\rangle + |111\rangle)^{\otimes 3} \text{ with phase encoding} \]

Conceptually: protect against \(X\) errors in one block, then protect against \(Z\) errors in a dual structure. Modern codes achieve the same goal more efficiently (Steane 7-qubit, surface codes).

Formal framework: Stabilizer Codes.

4) Syndrome measurement

Syndrome extraction measures commuting observables (stabilisers) using ancilla qubits. Outcomes form a bit string — the syndrome — mapping to the most likely error class.

  1. Couple code qubits to ancillas via CNOT or controlled gates
  2. Measure ancillas in \(Z\) basis — do not directly measure data qubits
  3. Classically decode syndrome → correction operator (Pauli)
  4. Apply correction or track in software (Pauli frame)
Faulty syndrome: If syndrome measurement itself errs, wrong corrections propagate errors. Fault-tolerant syndrome circuits are required — see Fault-Tolerant Quantum Computing.

5) Code parameters \([[n,k,d]]\)

A quantum code is described by:

  • \(n\): physical qubits
  • \(k\): logical qubits encoded
  • \(d\): distance — minimum weight of an undetectable error; corrects \(\lfloor(d-1)/2\rfloor\) errors

\[ \text{[[5,1,3]] surface patch} \quad \text{[[7,1,3]] Steane} \quad \text{[[9,1,3]] Shor} \]

Overhead: Large-scale factoring may need millions of physical qubits per logical qubit at achievable error rates — engineering drives code choice.

6) Threshold theorem (overview)

If physical gate error rate \(p\) is below a threshold \(p_{\mathrm{th}}\), arbitrarily long reliable computation is possible by increasing code size and concatenation or lattice surgery — error probability per logical operation can be driven arbitrarily small.

Estimates for surface codes place \(p_{\mathrm{th}}\) around \(10^{-3}\) to \(10^{-2}\) depending on assumptions. Current hardware often operates near or above these rates on two-qubit gates, motivating aggressive calibration and better codes.

Leading 2D implementation: Surface Codes.

7) Logical vs physical qubits

  • Physical qubit: raw device element (transmon, ion, etc.)
  • Logical qubit: protected degree of freedom of a code
  • Transversal gates: apply logical gates by physical gates in parallel — limited set
  • Magic state distillation: enables non-Clifford gates fault-tolerantly
NISQ contrast: NISQ Era Computing uses physical qubits with error mitigation instead of full QEC — sufficient only for short algorithms.
