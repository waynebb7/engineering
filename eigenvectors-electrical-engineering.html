Eigenvectors (Electrical Engineering)

Undergraduate guide. System modes, PCA preview, vibration mode shapes, and practical electrical engineering applications.

← Pure Math Subjects Catalog

1) Eigenvectors and mode directions

An eigenvector \(\mathbf{v}\) of matrix \(\mathbf{A}\) is a non-zero vector that satisfies \(\mathbf{A}\mathbf{v} = \lambda \mathbf{v}\). The eigenvalue \(\lambda\) scales the vector; the direction is unchanged.

For each eigenvalue, solve \((\mathbf{A} - \lambda\mathbf{I})\mathbf{v} = \mathbf{0}\). Any non-zero solution in the null space is an eigenvector (multiples of \(\mathbf{v}\) are equivalent).

Key idea: In engineering, eigenvectors describe mode shapes — the relative proportions of state variables during a natural response. Each mode evolves independently as \(e^{\lambda t}\) along its eigenvector direction.
Worked example: For \(\mathbf{A} = \begin{bmatrix} 0 & 1 \\ -4 & -3 \end{bmatrix}\) with \(\lambda = -1\): \((\mathbf{A} + \mathbf{I})\mathbf{v} = \mathbf{0}\) gives \(v_2 = -v_1\).
Eigenvector: \(\mathbf{v} = \begin{bmatrix} 1 \\ -1 \end{bmatrix}\) (or any scalar multiple).

Prerequisite: Eigenvalues (Electrical Engineering) and Eigenvectors.

2) State-space system modes

For \(\dot{\mathbf{x}} = \mathbf{A}\mathbf{x}\), the general solution (with \(n\) distinct eigenvalues) is: \[ \mathbf{x}(t) = \sum_{i=1}^{n} c_i\,e^{\lambda_i t}\,\mathbf{v}_i \] where \(c_i\) depend on initial conditions and \(\mathbf{v}_i\) are eigenvectors.

Physical meaning: Each term \(c_i e^{\lambda_i t}\mathbf{v}_i\) is a mode. If \(\mathbf{v}_1 = [1,\, -1]^T\) for an RLC circuit with states \([q,\, i]^T\), charge and current oscillate in opposite proportion during that mode.
Worked example: \(\mathbf{x}(0) = [2,\, 0]^T\), eigenvectors \(\mathbf{v}_1 = [1,\,1]^T\), \(\mathbf{v}_2 = [1,\,-1]^T\). Solve \(\mathbf{x}(0) = c_1\mathbf{v}_1 + c_2\mathbf{v}_2\) → \(c_1 = 1\), \(c_2 = 1\).
Response combines both modes with weights 1.
Common mistake: Using eigenvectors from the wrong matrix (open-loop vs closed-loop). Always match eigenvectors to the matrix governing the dynamics you are studying.

3) PCA preview — principal components in signal processing

Principal Component Analysis (PCA) applies eigenvector decomposition to the covariance matrix \(\mathbf{C}\) of sensor or signal data: \[ \mathbf{C}\mathbf{v}_i = \lambda_i \mathbf{v}_i \] Eigenvectors \(\mathbf{v}_i\) are principal components; eigenvalues \(\lambda_i\) give the variance along each direction.

Largest eigenvalue

Corresponding eigenvector captures the dominant signal direction — useful for dimensionality reduction and noise filtering.

Smallest eigenvalues

Often associated with noise subspaces. Truncating small-\(\lambda\) components removes low-energy noise.

EE application: In antenna array processing and EEG/EMG analysis, PCA eigenvectors identify coherent signal patterns. Retaining the top \(k\) components compresses data while preserving most energy: \(\dfrac{\sum_{i=1}^{k}\lambda_i}{\sum_{i=1}^{n}\lambda_i}\) of total variance.

4) Vibration modes in electrical machines

Mechanical vibration of motors and transformers satisfies: \[ \mathbf{M}\ddot{\mathbf{q}} + \mathbf{K}\mathbf{q} = \mathbf{0} \] Assuming \(\mathbf{q}(t) = \mathbf{Q}\,e^{j\omega t}\) leads to the generalised eigenvalue problem: \[ (\mathbf{K} - \omega^2\mathbf{M})\mathbf{Q} = \mathbf{0} \]

Mode shapes: Eigenvector \(\mathbf{Q}\) shows how each part of the machine moves relative to others at natural frequency \(\omega\). A mode might show rotor bending while the stator remains nearly still.
Design implication: Avoid operating frequencies near \(\omega\) where mode shapes couple strongly to electromagnetic forces — this causes resonance, noise, and mechanical failure.

Natural frequencies \(\omega\) correspond to eigenvalues of \(\mathbf{M}^{-1}\mathbf{K}\) (when \(\mathbf{M}\) is invertible).

5) RLC circuit mode shapes

For the series RLC state model with \(\mathbf{x} = [q,\, i]^T\), eigenvectors reveal how charge and current combine in each transient mode.

Underdamped case: Complex eigenvalues produce oscillatory modes. One eigenvector (and its conjugate) describes the phasor relationship between \(q\) and \(i\) during ringing after a switch event.
Worked example: \(\mathbf{A} = \begin{bmatrix} 0 & 1 \\ -100 & -10 \end{bmatrix}\), \(\lambda = -5 \pm j5\sqrt{3}\). For \(\lambda = -5 + j5\sqrt{3}\): \((\mathbf{A} - \lambda\mathbf{I})\mathbf{v} = \mathbf{0}\) yields \(v_2 = (-5 + j5\sqrt{3})\,v_1\).
The ratio \(i/q\) in that mode is complex — charge and current are 90° out of phase in the oscillatory part.

6) Diagonalisation and decoupled coordinates

If \(\mathbf{A}\) has \(n\) linearly independent eigenvectors, it is diagonalisable: \[ \mathbf{A} = \mathbf{P}\mathbf{\Lambda}\mathbf{P}^{-1}, \quad \mathbf{P} = [\mathbf{v}_1 \mid \mathbf{v}_2 \mid \cdots \mid \mathbf{v}_n], \quad \mathbf{\Lambda} = \text{diag}(\lambda_1, \ldots, \lambda_n) \] With \(\mathbf{z} = \mathbf{P}^{-1}\mathbf{x}\), the system decouples: \(\dot{z}_i = \lambda_i z_i\).

Engineering benefit: Decoupled modes simplify analysis — each \(z_i(t) = z_i(0)e^{\lambda_i t}\) independently. Control designers can target individual modes via state feedback.
Limitation: Repeated eigenvalues or defective matrices may not have enough independent eigenvectors. Use generalised eigenvectors or Jordan form in advanced courses.

7) Practical EE applications summary

Application Matrix / operator Eigenvector meaning
LTI circuit dynamics State matrix \(\mathbf{A}\) Relative voltage/current pattern per mode
Power grid oscillations Jacobian \(\mathbf{J}\) Generator angle/speed participation factors
Waveguide / cavity modes Maxwell operator Electric and magnetic field spatial patterns
PCA / dimensionality reduction Covariance \(\mathbf{C}\) Principal signal directions
Motor vibration \(\mathbf{K} - \omega^2\mathbf{M}\) Mechanical deformation shape at \(\omega\)
Coupled transmission lines Coupling matrix Even/odd mode voltage distributions

Eigenvectors answer how the system moves; eigenvalues answer how fast each mode grows or decays.
