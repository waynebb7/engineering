Differential Equations

Undergraduate guide. First-order separable and linear equations, second-order linear homogeneous ODEs, and applications.

← Pure Math Subjects Catalog

1) What is a differential equation?

A differential equation (DE) relates an unknown function and its derivatives. An ordinary differential equation (ODE) has one independent variable; a partial differential equation (PDE) has several (see Partial Differential Equations).

The order is the highest derivative present. A linear ODE has the form \[ a_n(x)y^{(n)} + a_{n-1}(x)y^{(n-1)} + \cdots + a_1(x)y' + a_0(x)y = g(x) \] Otherwise it is nonlinear.

Key idea: A general solution contains arbitrary constants equal to the order of the ODE. An initial value problem (IVP) fixes \(y, y', \ldots\) at a point to determine those constants uniquely.

Builds on Differentiation, Integration, and Functions.

2) First-order separable equations

A first-order ODE is separable if it can be written as: \[ \frac{dy}{dx} = f(x)\,g(y) \quad \Leftrightarrow \quad \frac{1}{g(y)}\,dy = f(x)\,dx \] Integrate both sides and solve for \(y\) if possible.

Worked example: Solve \(\dfrac{dy}{dx} = xy\).
Separate: \(\dfrac{dy}{y} = x\,dx\). Integrate: \(\ln|y| = \dfrac{x^2}{2} + C\).
General solution: \(y = Ae^{x^2/2}\) where \(A = \pm e^C\).
Worked example: Solve \(\dfrac{dy}{dx} = y^2\), \(y(0) = 1\).
\(\dfrac{dy}{y^2} = dx \Rightarrow -\dfrac{1}{y} = x + C\). With \(y(0)=1\): \(C = -1\).
Solution: \(y = \dfrac{1}{1-x}\), valid for \(x < 1\) (blows up at \(x = 1\)).
Common mistake: Dividing by \(g(y)\) without checking whether constant solutions \(g(y) = 0\) were lost.

3) First-order linear equations

A first-order linear ODE has the form: \[ \frac{dy}{dx} + P(x)y = Q(x) \] Multiply by the integrating factor \(\mu(x) = e^{\int P(x)\,dx}\): \[ \frac{d}{dx}[\mu(x)\,y] = \mu(x)\,Q(x) \]

Worked example: Solve \(y' + 2y = 6e^{-x}\).
\(\mu = e^{\int 2\,dx} = e^{2x}\). Then \(\dfrac{d}{dx}(e^{2x}y) = 6e^{x}\).
Integrate: \(e^{2x}y = 6e^x + C\), so \(y = 6e^{-x} + Ce^{-2x}\).

Homogeneous case

When \(Q(x) = 0\): \(y = Ce^{-\int P\,dx}\).

Bernoulli equations

\(y' + P(x)y = Q(x)y^n\) — substitute \(v = y^{1-n}\) to linearise.

4) Second-order linear homogeneous ODEs

The second-order linear homogeneous equation with constant coefficients is: \[ ay'' + by' + cy = 0, \quad a \neq 0 \] Substitute \(y = e^{rx}\) to obtain the characteristic equation: \[ ar^2 + br + c = 0 \]

Roots of \(ar^2 + br + c = 0\) General solution
Distinct real \(r_1, r_2\) \(y = C_1 e^{r_1 x} + C_2 e^{r_2 x}\)
Repeated real \(r\) (double root) \(y = (C_1 + C_2 x)e^{rx}\)
Complex \(\alpha \pm i\beta\) \(y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)\)
Worked example: Solve \(y'' - 5y' + 6y = 0\).
Characteristic: \(r^2 - 5r + 6 = (r-2)(r-3) = 0\), so \(r = 2, 3\).
General solution: \(y = C_1 e^{2x} + C_2 e^{3x}\).

5) Repeated and complex roots in detail

When the characteristic equation has a repeated root \(r\), \(e^{rx}\) and \(xe^{rx}\) are linearly independent solutions (the second arises from the limit as two distinct roots coalesce).

Worked example: Solve \(y'' - 4y' + 4y = 0\).
\((r-2)^2 = 0\), so \(r = 2\) (double). General solution: \(y = (C_1 + C_2 x)e^{2x}\).

For complex roots \(\alpha \pm i\beta\), Euler's formula gives real-valued solutions. The amplitude-phase form \(y = Ae^{\alpha x}\cos(\beta x - \phi)\) is often used in engineering.

Worked example: Solve \(y'' + 4y = 0\).
\(r^2 + 4 = 0 \Rightarrow r = \pm 2i\). General solution: \(y = C_1\cos 2x + C_2\sin 2x\).
Exam tip: For IVPs, apply initial conditions to the general solution before rewriting in amplitude-phase form.

6) Applications

Exponential growth and decay

\(y' = ky\) models population growth (\(k>0\)) or radioactive decay (\(k<0\)): \(y = y_0 e^{kt}\).

Newton's law of cooling

\(T' = -k(T - T_{\text{env}})\) — separable; solution approaches ambient temperature.

Simple harmonic motion

\(my'' + ky = 0\) gives \(y = A\cos(\omega t) + B\sin(\omega t)\) with \(\omega = \sqrt{k/m}\).

RLC circuits

\(L\dfrac{d^2q}{dt^2} + R\dfrac{dq}{dt} + \dfrac{q}{C} = 0\) — overdamped, critically damped, or underdamped depending on the discriminant \(R^2 - 4L/C\).

Worked example (mixing): A tank holds 100 L of brine with 10 kg salt. Fresh water enters at 5 L/min; well-mixed solution leaves at 5 L/min. Let \(S(t)\) = salt mass.
\(S' = -\dfrac{5}{100}S = -0.05S\), \(S(0) = 10\). Solution: \(S(t) = 10e^{-0.05t}\) kg.

7) Initial value problems and linear independence

For a second-order linear ODE, two solutions \(y_1, y_2\) are linearly independent if the Wronskian \(W = y_1 y_2' - y_1' y_2\) is not identically zero. The general solution is then \(y = C_1 y_1 + C_2 y_2\).

Worked example (IVP): Solve \(y'' + y = 0\), \(y(0) = 1\), \(y'(0) = 0\).
General: \(y = C_1\cos x + C_2\sin x\). From \(y(0)=1\): \(C_1 = 1\). From \(y'(0)=0\): \(C_2 = 0\).
Solution: \(y = \cos x\).

Existence and uniqueness: For \(y' = f(x,y)\) with \(y(x_0) = y_0\), if \(f\) and \(\partial f/\partial y\) are continuous near \((x_0, y_0)\), a unique local solution exists (Picard–Lindelöf theorem).

Non-homogeneous equations \(ay'' + by' + cy = g(x)\) require particular solutions (undetermined coefficients, variation of parameters) — covered in advanced ODE courses.
